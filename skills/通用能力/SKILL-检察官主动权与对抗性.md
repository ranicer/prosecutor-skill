# 检察官主动权与对抗性

> 检察官不是被动的"审查者"，而是拥有法定职权的"进攻者"。本模块系统梳理检察官在办案各环节中如何行使主动权——对公安能施压推进、对律师能有效应对、对法院能守住底线并反制不当判决，在法庭辩论中掌握进攻节奏。

---

## 一、检察官的天然优势（常被忽视）

### 1.1 五大法定优势

| 优势 | 具体内容 | 实务运用 |
|------|---------|---------|
| **程序控制权** | 退补、延长审限、不起诉决定 | 掌握案件节奏，公安和律师都被动配合 |
| **法律监督权** | 对侦查活动、审判活动的监督 | 对公安违法取证、法院程序违法提出纠正 |
| **抗诉权** | 对不当判决的终极反制 | 法院不敢轻易无视检察院的抗诉 |
| **认罪认罚主导权** | 量刑建议权在检察院 | 协商中检察官是主导方，不是被协商方 |
| **信息优势** | 全案卷宗在检察官手中 | 公安和律师都是"局部视角"，检察官是"全局视角" |

### 1.2 核心定位

**检察官的"强"不是态度上的强，而是：**
- **专业上的强**：证据分析精准、法律适用准确、说理充分
- **程序上的强**：严格依法办案，让对方无程序漏洞可钻
- **证据上的强**：全案证据在手，构建完整的证据体系
- **策略上的强**：知道什么时候进攻、什么时候防守、什么时候让步

## 二、对公安的施压与推进

### 2.1 公安"软抵抗"的系统反制

#### 退补后消极补查——五步反制法

```
第一步：退补提纲写到"极致明确"
  - 具体到证据名称、存放位置、证明目的
  - 附："如无法补查，将因证据不足不起诉"
  ↓
第二步：退补后第10天主动催办
  - 电话/当面询问进展
  - 要求提供补查进度说明
  ↓
第三步：发《补充侦查催办函》（留痕）
  - 书面催办，附退补提纲要求的具体事项
  - 抄送公安法制部门
  ↓
第四步：二次退补仍不达标
  - 制作《补查情况评估报告》
  - 逐项对比：要求什么 → 实际补了什么 → 差距在哪
  - 提交检察官联席会议讨论
  ↓
第五步：依法作存疑不起诉
  - 不起诉决定书详细列明补查不到位的事实
  - 抄送公安法制部门
  - 作为案件质量问题通报
```

**核心原则：绝对不能因为公安补查不到位就勉强起诉。宁可作存疑不起诉，也不能带病起诉。**

#### 公安不配合立案监督——三级递进

| 级段 | 手段 | 时限 |
|------|------|------|
| 一级 | 书面要求公安说明立案情况 | 15日内 |
| 二级 | 理由不成立的，层报上级检察院 | 即时 |
| 三级 | 向同级人大常委会报告 | 必要时 |

#### 公安材料不齐全——"门都不让进"

- 重要证据缺失：发《补充材料函》，**不进入审查程序**
- 程序性材料缺失：记录在案，要求补正
- 制作《案件材料核对清单》，逐项勾选

### 2.2 检察官的"主动出击"手段

#### A. 提前介入侦查
- **适用**：重大复杂案件、新型犯罪、社会关注度高的案件
- **操作**：提出书面介入意见，明确需要收集的证据清单
- **价值**：从源头把控证据质量，避免后期被动

#### B. 自行补充侦查
- **适用**：退补后仍不达标，但案件有起诉价值
- **优势**：不依赖公安，自己动手
- **操作**：检察官亲自或指派助理调取证据、询问证人

#### C. 侦查活动监督——对公安最直接的制约

| 监督对象 | 手段 | 实际效果 |
|---------|------|---------|
| 违法取证 | 《纠正违法通知书》 | 纳入公安绩效考核 |
| 超期羁押 | 提出纠正意见 | 影响办案人员考评 |
| 程序违法 | 口头纠正 + 书面记录 | 留痕备查 |

### 2.3 对公安施压的话术

**补查材料敷衍时**：
> "我看了你们补查回来的材料，但是×××这个关键问题还是没有解决。退补提纲里明确要求调取×××，现在调回来的材料还是原来的那些，没有新内容。这个案子如果补查不到位，可能要作存疑不起诉处理。希望你们再认真补查一下。"

**公安要求延长羁押但理由不充分时**：
> "延长拘留的法定条件是'流窜作案、多次作案、结伙作案'。这个案子目前的情况，×××是否符合这些条件？请你们补充说明具体的延长理由，不能笼统写'案情复杂'。"

**公安以"上面有压力"要求起诉时**：
> "我理解你们的压力。但检察院办案有独立的审查标准，证据不足的案子我们不能勉强起诉。如果起诉后判无罪，承办检察官要被追责的。我建议你们再补查一下×××这个关键事实。"

## 三、对律师的应对与反制

### 3.1 律师常见策略及反制

#### "死磕派"律师——不被激怒是第一要务

**律师策略**：当庭大声质问、程序性挑刺、舆论施压、反复纠缠

**检察官反制**：
1. **情绪控制**：不被激怒，一旦情绪化你就输了
2. **专业回应**：所有质疑用证据和法律回应，不带情绪
3. **书面留痕**：口头沟通后形成书面记录，防止被歪曲
4. **场所控制**：所有正式沟通在办公场所进行（律师可能录音）
5. **借助程序**：律师行为严重影响办案时，向司法行政部门反映

**具体场景应对**：

| 场景 | 正确应对 | 错误应对 |
|------|---------|---------|
| 律师当庭大声质问 | 保持沉默，等待审判长处理 | 与律师对吵 |
| 律师反复纠缠同一问题 | "已充分回应，记录在案，如有新观点请书面提交" | 反复解释 |
| 律师在网上发布不实信息 | 不直接回应，向院领导和宣传部门报告 | 在网上对骂 |
| 律师当庭人身攻击 | 当庭简要回应，庭后书面提出 | 当庭反击 |

#### 律师拖延战术——用程序规则反制

| 律师拖延手段 | 检察官反制 |
|------------|-----------|
| 反复申请阅卷 | 依法保障，但不做不必要的配合 |
| 申请排除非法证据 | 要求提供初步线索和材料（不能空口白牙） |
| 申请证人出庭 | 审查必要性，不必要的依法驳回 |
| 申请重新鉴定 | 审查必要性，无正当理由的驳回 |
| 反复变更辩护人 | 依法保障权利，但记录在案 |

#### 律师教唆翻供——事后追究

- 庭审中揭露翻供的不合理性
- 庭后向司法行政部门反映
- 固定相关证据（如律师会见记录等）

### 3.2 认罪认罚协商中的主动权

#### 锚定效应——谁先出价谁主动

- **先发制人**：检察官先提出量刑建议，让辩护人在你的框架内讨价还价
- **不要等律师先出价**：一旦律师先出价，你就被动了
- **量刑建议附详细说理**：增加说服力，减少讨价还价空间

#### 最后通牒策略——设定协商窗口期

> "本周内达成一致，量刑建议为×××；如果本周未达成，量刑建议将调整为×××。"

- 利用嫌疑人急于从宽的心理
- 所有协商内容书面固定
- 防止夜长梦多

#### 协商破裂的后备方案

- 检察官有**不起诉权**——这是最大的筹码
- 即使协商不成，检察官仍有多种处理方式
- 不怕破裂——怕破裂的是律师和嫌疑人

### 3.3 对律师施压的话术

**律师漫天要价时**：
> "您的意见我们理解，但从案件事实和法律规定来看，确实有差距。我们不妨各自把依据说清楚。我们的底线是×××，您看怎么样？"

**律师反复变卦时**：
> "今天的协商内容已经书面固定了。认罪认罚从宽的量刑优惠是有条件的，如果最终没有达成一致，量刑建议会相应调整。"

**律师威胁舆论施压时**：
> "我们依法办案，欢迎监督。如果辩护人认为有违法情形，可以通过正式渠道反映。"

## 四、对法院的反制能力

### 4.1 量刑建议不被采纳——四级应对

```
第一级：庭审前预防
  - 量刑建议充分说理，附详细计算过程
  - 提前与法官非正式沟通
  - 量刑建议留有适当弹性
  ↓
第二级：庭审中应对
  - 法院当庭告知不采纳 → 要求说明理由并记录在案
  - 根据庭审情况调整量刑建议
  - 充分发挥公诉人对庭审的引导作用
  ↓
第三级：庭后沟通（黄金窗口期）
  - 庭审结束后1-3天内联系法官
  - 重申量刑建议的事实和法律依据
  - 了解法官的初步判断和分歧点
  - 针对分歧点补充说明或调整
  ↓
第四级：判决后抗诉
  - 审查判决书的量刑说理
  - 比较量刑建议与判决结果的差距
  - 差距明显不合理的，依法抗诉
```

**庭后沟通话术**：
> "张法官，关于×××案的量刑，我想再跟您沟通几点。我们提出建议判处×××是有依据的：第一，根据量刑指导意见，基础刑期是……；第二，有×××从轻情节，减少……；第三，有×××从重情节，增加……。综合下来，建议刑期在×××到×××之间。不知道您对量刑有什么考虑？"

**法律依据**：刑诉法第201条——法院一般应当采纳量刑建议，不采纳的应当说明理由。

### 4.2 法院程序违法——三种纠正方式

| 发现时间 | 纠正手段 | 操作要点 |
|---------|---------|---------|
| 庭审中发现 | 当庭口头纠正 | 要求书记员记录在案 |
| 庭审后发现 | 《纠正审理违法意见书》 | 在法定期限内提出 |
| 判决后发现 | 作为抗诉理由之一 | 在抗诉书中详细列明 |

### 4.3 法院消极审理——主动推进

| 情形 | 反制手段 |
|------|---------|
| 超期审理 | 发函催办 + 层报上级 |
| 未依法通知出庭 | 记录在案 + 提出纠正意见 |
| 审判组织不合法 | 当庭提出 |
| 庭审走过场 | 充分发挥公诉人引导作用 |

### 4.4 抗诉——终极反制武器

**抗诉前评估**：
1. 判决是否确有错误（事实认定、法律适用、量刑）
2. 抗诉成功率如何
3. 社会效果如何
4. 对检法关系的长期影响

**抗诉的威慑力**：
- 抗诉是检察院最有力的监督手段
- 法院对检察院的抗诉高度重视
- 即使抗诉后改判率不高，抗诉本身就有威慑作用

## 五、法庭辩论的进攻性技巧

### 5.1 公诉人庭审的"进攻性"定位

**核心理念**：公诉人不是被动的"举证方"，而是庭审的**主导者**。

- 通过讯问被告人、举证质证、答辩辩护意见三个环节，全面展示控方证据体系
- 目标：让法官形成"指控成立"的心证
- 进攻是最好的防守——主动出击比被动回应更有效
- **"检察院两把剑，插完公安插法院"**——检察官同时监督侦查和审判，天然具有双重主动权

### 5.2 被告人讯问——锁定供述的四步法

#### A. 封闭式提问法（最基础也最有效）

- 用"是或不是""有没有""是不是这样"限制回答范围
- 不给被告人"讲故事"的机会
- 在关键事实上连续追问，形成"供述锁链"

**话术示例**：
> "我问你几个问题，你如实回答。2024年3月5日晚上，你是不是在×××？——是。你是不是认识被害人×××？——是。你是不是在那天晚上和被害人发生了冲突？——是。你是不是动手打了被害人？——是。被害人倒地后，你是不是又踢了他？——是。"

#### B. 矛盾揭露法

- 当庭指出被告人供述与此前供述的矛盾
- 逐条对比，让矛盾暴露在法官面前
- 追问翻供原因，揭示不合理性

**话术示例**：
> "被告人在2024年3月8日、3月15日、4月1日的三次供述中，均稳定供述了以下事实：第一……第二……今天庭审中你说这些供述不属实。请法庭注意，三次供述的主要内容是一致的。被告人，请你解释一下，为什么之前三次供述都不属实？"

#### C. 证据突击法

- 在被告人以为检察院没有关键证据时，突然出示
- 观察其反应，趁势追问
- 用客观证据（监控、DNA、转账记录）锁定犯罪事实

**操作要点**：
1. 先让被告人充分表达"无罪"版本
2. 固定其供述后再出示关键证据
3. 追问其如何解释客观证据
4. 不给其"圆谎"的机会

#### D. 情感与法律结合

- 在讯问中适当表达对被害人处境的关注
- 指出被告人的行为造成的实际危害
- 让法官和社会感受到犯罪的危害性

#### 法庭讯问10个要点（实务归纳）

| 要点 | 内容 | 注意事项 |
|------|------|---------|
| 1. 闭合式提问为主 | "是或不是""有没有" | 不让被告人自由发挥 |
| 2. 由浅入深 | 先问无争议事实，再问关键事实 | 建立讯问节奏 |
| 3. 封锁退路 | 先固定细节，再问核心问题 | 让被告人无法"圆谎" |
| 4. 利用矛盾 | 当庭指出供述与此前笔录的矛盾 | 逐条对比，不留情面 |
| 5. 控制语速 | 关键问题放慢语速，加重语气 | 给法官充分记录时间 |
| 6. 不要重复 | 同一问题不要反复问 | 反复问等于给被告人"修正"机会 |
| 7. 不要解释 | 问完问题等回答，不要自己补充 | 补充解释等于提示被告人 |
| 8. 紧盯眼睛 | 讯问时注视被告人 | 观察其微表情和反应 |
| 9. 不要吵架 | 被告人反驳时不要争辩 | 用证据回应，不用情绪回应 |
| 10. 适时收手 | 关键事实锁定后及时停止 | 不要画蛇添足给辩护人留下攻击点 |

### 5.3 举证质证——构建证据体系

**举证原则**：
- 按逻辑顺序出示证据，不按时间顺序
- 先出示基础性证据（身份、前科等）
- 再出示直接证据（供述、证言等）
- 最后出示核心证据（物证、鉴定意见等）
- 每组证据之间有逻辑衔接

**举证话术**：
> "公诉人现就本案证据分组出示。第一组：证明被告人身份和前科情况的证据……第二组：证明犯罪事实的证据……第三组：证明量刑情节的证据……"

### 5.4 答辩辩护意见——逐点回应，反守为攻

#### 基本原则

1. **逐点回应，不遗漏**：辩护人的每一个观点都要回应，不回应 = 默认
2. **先破后立**：先指出辩护观点的错误，再阐述控方观点
3. **反守为攻**：不只被动答辩，还要主动追问
4. **简洁有力**：每点回应控制在1-2分钟

#### 应对不同辩护策略

| 辩护策略 | 公诉人应对 |
|---------|-----------|
| **质疑证据合法性** | 出示同步录音录像、讯问笔录等取证合法的证明 |
| **质疑证据关联性** | 论证证据与待证事实的逻辑联系 |
| **提出无罪辩护** | 回归事实和证据，逐一驳斥 |
| **提出罪轻辩护** | 说明量刑建议已充分考虑相关情节 |
| **质疑鉴定意见** | 申请鉴定人出庭或提供补充说明 |
| **提出非法证据排除** | 要求提供初步线索 + 出示取证合法性证据 |

#### 反守为攻话术

**追问辩护人**：
> "辩护人提出×××观点，但公诉人想请问：如果按照辩护人的逻辑，那×××怎么解释？×××证据又如何理解？"

**指出辩护逻辑漏洞**：
> "辩护人一方面认为证据不足，另一方面又建议从轻处罚。如果证据不足，应该是无罪辩护而非罪轻辩护。辩护人的逻辑是矛盾的。"

**回到证据和法律**：
> "辩护人的观点很有启发性，但法律是讲证据的。本案有以下证据能够印证：第一……第二……这些证据形成了完整的证据链条。辩护人的辩护意见不能成立。"

### 5.5 公诉意见——庭审的"收官之作"

**结构建议**：

```
一、关于事实认定
  - 概述案件事实
  - 强调证据体系的完整性

二、关于法律适用
  - 指出被告人行为构成何罪
  - 详细论证犯罪构成要件

三、关于量刑建议
  - 列明从重/从轻情节
  - 说明量刑建议的依据

四、关于辩护意见的回应
  - 逐点回应辩护观点
  - 强调辩护意见不能成立

五、结论
  - 请合议庭依法采纳量刑建议
```

**话术示例**：
> "审判长、审判员：
>
> 受×××人民检察院指派，我代表本院以国家公诉人的身份出席法庭。
>
> 通过刚才的法庭调查，本案事实清楚，证据确实、充分。现就本案发表如下意见：
>
> 第一，关于事实认定：……
>
> 第二，关于法律适用：被告人的行为构成×××罪，依据×××法第×××条……
>
> 第三，关于量刑建议：综合考虑×××等情节，建议判处×××。
>

## 六、灰色操作反制能力

### 6.1 对公安灰色操作的反制

| 公安灰色操作 | 检察官反制手段 |
|------------|--------------|
| 笔录"加工"（修正口语化表达使其像有罪供述） | 核查同步录音录像，对比笔录与录音的差异 |
| 选择性记录（只记有罪部分，忽略辩解） | 审查供述完整性，要求调取全部讯问录像 |
| 事后补正笔录 | 审查笔录时间戳和修改痕迹 |
| "先抓人后找证据" | 严格审查证据标准，不符合标准的依法不起诉 |
| 技术侦查扩大使用 | 审查技术侦查审批手续的合法性 |
| 通过关系施压要求起诉 | 坚持证据标准底线，书面记录压力来源 |
| 笔录与录音录像不符 | 以录音录像为准，排除相关笔录 |

### 6.2 对法院灰色操作的反制

| 法院灰色操作 | 检察官反制手段 |
|------------|--------------|
| 未审先判（庭审走过场） | 充分发挥公诉人引导作用，主动申请讯问和举证 |
| 不采纳量刑建议且不说明理由 | 要求法院依法说明理由（刑诉法201条） |
| 超期审理 | 催办 + 层报 + 记录在案 |
| 违法缺席判决 | 当庭提出纠正意见 |
| 判决书不说理 | 以量刑说理不充分为由考虑抗诉 |
| 审判组织不合法 | 当庭提出 |

### 6.3 检察官自我保护

1. **全程留痕**：所有重要沟通书面化
2. **重大案件上会**：集体讨论决定，不一个人扛
3. **依法办案**：只要依法，就不怕追责
4. **记录领导指示**：领导说的每句话都可能成为追责依据
5. **不越权**：助理不做检察官的决定
6. **供述自愿性核查**：每次讯问后确认嫌疑人供述自愿性，留痕备查

## 八、实战案例

### 案例一：零口供突破——检察官如何在没有被告人供述的情况下定罪

**案件背景**：某污染环境案（5千吨毒污泥），被告人始终拒不认罪。

**检察官的主动出击策略**：
1. **证据体系构建**：不依赖口供，构建以客观证据为主的证据链
   - 环境监测数据
   - 转运记录和合同
   - 被害单位的证言
   - 被告人的银行流水
2. **讯问策略**：虽然被告人不认罪，但通过讯问暴露其辩解的矛盾
   - 让被告人充分表达"无罪"版本
   - 用客观证据逐条反驳
   - 固定其辩解内容，为庭审做准备
3. **庭审效果**：被告人当庭翻供，但检察官用完整的客观证据链说服法官

**启示**：即使没有口供，只要客观证据体系完整，检察官依然可以成功指控。关键在于不依赖口供、主动构建证据体系。

### 案例二：认罪认罚协商中的主导权——检察官如何掌控协商节奏

**常见场景**：辩护人认为量刑建议偏重，要求调整。

**检察官的主导策略**：
1. **先发制人**：检察官先提出量刑建议，设定协商框架
2. **展示证据优势**：让辩护人知道"硬辩没有空间"
3. **利益引导**：详细解释认罪认罚从宽的具体幅度
4. **设定窗口期**：明确告知协商时限，推动决策
5. **书面固定**：所有协商内容即时书面记录

**关键话术**：
> "我们的量刑建议是基于×××的考虑。如果被告人能够退赃退赔并取得被害人谅解，我们可以适当调整。但如果本周内没有达成一致，量刑建议将相应调整。"

## 九、重要提示

1. **检察官的主动权来自法律授权**，不是个人权力。行使主动权必须在法律框架内。
2. **进攻不等于霸道**。检察官的"强"是专业上的强、证据上的强、程序上的强，不是态度上的强。
3. **客观公正义务不可放弃**。检察官既是控诉者，也是法律监督者。发现有利于被告人的证据，依法客观处理。
4. **保护好自己**：全程留痕、重大案件上会、依法办案。
5. **认罪认罚的主导地位**：量刑建议权是检察官的核心武器，要善于运用。
