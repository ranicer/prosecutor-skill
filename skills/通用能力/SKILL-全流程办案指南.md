# 全流程办案指南

本 Skill 是检察官助理的**总操作手册**，以一个普通刑事案件为例，串起从收到案件到归档结案的每一个步骤，标注每个节点的具体操作、产出文书、期限要求和易错点。

---

## 全流程总览

```
案件受理 → 审查逮捕(如适用) → 审查起诉 → 出庭支持公诉 → 判决审查 → 执行监督 → 归档结案
```

---

## 阶段一：案件受理（收到当天）

### 1.1 接收材料

**操作**：从案管部门领取案件卷宗，核对移送材料。

**核对清单**：

```
□ 起诉意见书（公安机关移送）
□ 立案决定书
□ 拘留证 / 延长拘留期限决定书
□ 逮捕证 / 变更强制措施文书
□ 犯罪嫌疑人讯问笔录（全部，含有罪供述和辩解）
□ 被害人陈述笔录
□ 证人证言笔录
□ 物证、书证（扣押清单、照片）
□ 鉴定意见（伤情鉴定、价格认定、DNA等）
□ 勘验/检查笔录
□ 辨认笔录
□ 视听资料、电子数据（光盘/U盘）
□ 搜查证、搜查笔录
□ 犯罪嫌疑人权利义务告知书
□ 前科劣迹材料（判决书、释放证明等）
□ 认罪认罚相关材料（如有）
□ 涉案财物清单（如有）
□ 逮捕必要性说明（如有）
```

**易错点**：
- 被告人供述只有有罪供述，无辩解笔录 → 记录，后续讯问时补充
- 关键证据缺失（如被害人陈述、鉴定意见）→ 发《补充材料函》
- 讯问笔录侦查人员只签一人 → 记录，要求补正

### 1.2 登记与分案

**操作**：
1. 在统一业务应用系统中登记案件
2. 确认承办人（自己或带教检察官）
3. 记录收案日期（期限起算点）

### 1.3 初步阅卷（30分钟）

**目的**：快速了解案件全貌，判断案件类型和复杂程度。

**操作**：
1. 先看**起诉意见书**（了解公安认定的事实和罪名）
2. 再看**讯问笔录**（了解嫌疑人供述和辩解情况）
3. 快速翻阅**鉴定意见**（伤情/价格等关键证据）
4. 初步判断：事实是否清楚？证据是否充分？有无明显问题？

**产出**：在笔记本上记录初步印象和疑问。

## 阶段二：审查逮捕（如公安机关提请，7日内完成）

### 2.1 讯问犯罪嫌疑人（必须）

**时机**：收到提请后1-2日内安排讯问

**讯问提纲模板**：

```
一、身份核实
  姓名、出生日期、身份证号、民族、文化程度、住址、职业

二、告知权利
  "你有权委托辩护人，如符合法律援助条件，可以申请法律援助。
   你有权拒绝回答与本案无关的问题。"

三、案件事实核实
  1. 你是什么时间、地点被抓获的？
  2. 你因什么事被刑事拘留？
  3. （按起诉意见书认定的事实逐项讯问）
  4. 你对起诉意见书认定的事实有无异议？
  5. 你是否认罪认罚？

四、强制措施意见
  1. 你是否愿意取保候审？
  2. 你有无固定住所？
  3. 你有无保证人/保证金？
  4. 你是否有可能逃跑、串供、毁灭证据？

五、特殊事项
  1. 你身体状况如何？有无严重疾病？
  2. 你是否怀孕或正在哺乳？
  3. 你有无自杀倾向？
  4. 你有无其他需要说明的情况？
```

**讯问笔录制作要点**：
- 全程记录，不遗漏关键问答
- 犯罪嫌疑人辩解的内容必须完整记录
- 讯问结束后交嫌疑人核对签字
- 记录讯问起止时间、地点、参与人员

### 2.2 听取辩护人/值班律师意见

**操作**：
- 有辩护人的，联系辩护人听取意见
- 无辩护人的，联系值班律师，告知认罪认罚权利
- 记录辩护人/值班律师的意见

### 2.3 证据审查（核心，2-4小时）

按以下顺序逐项审查：

**第一步：程序合法性审查**
```
□ 讯问笔录是否有两名以上侦查人员签名
□ 讯问是否告知权利义务
□ 讯问时间是否存在连续超过12小时的情况
□ 拘留/逮捕是否在法定期限内
□ 延长羁押期限是否合法
□ 鉴定意见是否告知犯罪嫌疑人
□ 辩护权是否得到保障
```

**第二步：证据三性审查**
```
合法性：
□ 物证、书证的提取是否符合法定程序
□ 讯问是否存在刑讯逼供等非法取证
□ 视听资料的来源是否合法
□ 鉴定机构和鉴定人是否具有资质

真实性：
□ 记忆笔录内容是否前后一致
□ 证人与当事人有无利害关系
□ 鉴定检材来源是否可靠
□ 电子数据提取是否规范

关联性：
□ 证据与案件事实是否有关联
□ 各证据之间能否相互印证
□ 有无无法排除的矛盾
```

**第三步：制作证据清单表格**

| 序号 | 种类 | 名称 | 证明事项 | 页码 | 合法 | 真实 | 关联 | 问题 |
|------|------|------|---------|------|------|------|------|------|
| 1 | 书证 | 户籍证明 | 主体身份 | 12 | ✓ | ✓ | ✓ | |
| 2 | 鉴定 | 伤情鉴定 | 轻伤二级 | 45 | ✓ | ✓ | ✓ | |
| 3 | 证言 | 张三证言 | 案发经过 | 30 | ✓ | ? | ✓ | 与李四证言矛盾 |
| ... | | | | | | | | |

### 2.4 社会危险性评估

按五项标准逐项评估（详见 `skills/刑事检察/SKILL-审查逮捕.md`）。

### 2.5 制作审查意见 → 提请批准/不批准逮捕

**产出文书**：
- 批准逮捕：制作《审查逮捕意见书》
- 不批准逮捕：制作《不批准逮捕理由说明书》

## 阶段三：审查起诉（核心阶段，1个月+可延长15日）

### 3.1 全案阅卷（结构化阅卷法，4-8小时）

**五步阅卷法**（详见 `skills/通用能力/SKILL-阅卷与出庭技巧.md`）：

```
Step 1：通览全卷（30分钟）
  → 了解全貌，标记重点

Step 2：程序审查（1小时）
  → 核查所有程序性文书，标注期限问题

Step 3：事实审查（2-4小时）
  → 制作案件时间轴，梳理犯罪构成

Step 4：证据审查（2-4小时）
  → 制作证据清单+证据矩阵

Step 5：形成阅卷笔录（1小时）
  → 汇总发现的问题和下一步方向
```

**阅卷笔录必须记录的内容**：

```
1. 案件基本信息（当事人、罪名、作案时间）
2. 各被告人供述要点（有罪供述+辩解）
3. 被害人陈述要点
4. 证人证言要点（按证明事项分类）
5. 物证、书证清单
6. 鉴定意见要点（鉴定结论、鉴定资质）
7. 证据之间的印证关系
8. 证据矛盾和疑点
9. 程序性问题
10. 需要补充的证据
11. 初步处理意见
```

### 3.2 讯问犯罪嫌疑人（必须）

**与审查逮捕阶段讯问的区别**：
- 审查起诉阶段讯问更详细，需核实全部犯罪事实
- 重点核实阅卷中发现的疑点和矛盾
- 告知认罪认罚的法律后果
- 听取嫌疑人对案件处理的意见

**讯问提纲（审查起诉版）**：

```
一、身份核实（同上）

二、权利告知
  "你有权委托辩护人。你涉嫌×××罪，可能判处×年以下有期徒刑。
   如你自愿认罪认罚，可以依法从宽处理。"

三、犯罪事实核实（逐项核实）
  （按起诉意见书认定的每项犯罪事实逐一讯问）
  （对阅卷中发现的疑点重点讯问）
  （对嫌疑人辩解的内容详细记录）

四、量刑情节核实
  1. 你有无自首情节？（自动投案+如实供述）
  2. 你有无检举揭发他人犯罪？
  3. 你是否赔偿被害人？取得谅解了吗？
  4. 你的家庭情况如何？（是否有需要抚养的人）
  5. 你对案件处理有什么要求？

五、认罪认罚
  1. 你是否自愿认罪？
  2. 你是否同意起诉书指控的犯罪事实？
  3. 你是否愿意接受处罚？
  4. 你是否理解认罪认罚的法律后果？
  5. 你是否同意签署认罪认罚具结书？
```

### 3.3 听取被害人意见

**操作**：
- 联系被害人（或其诉讼代理人）
- 告知案件进展
- 听取对案件处理的意见
- 了解赔偿/谅解情况
- 记录被害人诉求

**注意事项**：
- 被害人情绪激动时做好安抚
- 不承诺具体处理结果
- 做好接待记录

### 3.4 听取辩护人意见

**操作**：
- 联系辩护人（或值班律师）
- 交换案件意见
- 了解辩护观点
- 协商认罪认罚量刑建议

**协商要点**：
- 案件事实和定性
- 量刑情节的认定
- 量刑建议的幅度
- 是否适用缓刑
- 程序适用（速裁/简易/普通）

### 3.5 犯罪构成分析 + 证据综合审查

详见 `skills/通用能力/SKILL-案件分析.md` 和 `skills/通用能力/SKILL-证据整合与量刑辅助.md`。

### 3.6 量刑情节分析 + 量刑建议计算

详见 `skills/通用能力/SKILL-证据整合与量刑辅助.md` 中的量刑计算表。

### 3.7 处理决定

```
┌─────────────────────────────────────────────────┐
│              审查起诉处理决定                      │
├────────────┬────────────┬───────────┬────────────┤
│  提起公诉   │ 不起诉      │ 退回补充侦查 │ 自行补充侦查 │
│            │            │           │            │
│ 事实清楚    │ 法定不起诉  │ 以2次为限   │ 每次1个月   │
│ 证据确实充分 │ 酌定不起诉  │ 每次1个月   │            │
│ 依法应追究   │ 存疑不起诉  │ 重新计算期限 │            │
│ 刑事责任    │ 附条件不起诉 │            │            │
└────────────┴────────────┴───────────┴────────────┘
```

### 3.8 制作文书

**根据处理决定制作对应文书**：

| 处理决定 | 产出文书 | 文书格式化命令 |
|---------|---------|--------------|
| 提起公诉 | 起诉书 | `python scripts/文书格式化.py 起诉书 data.json` |
| 法定不起诉 | 不起诉决定书 | `python scripts/文书格式化.py 不起诉决定书 data.json` |
| 酌定不起诉 | 不起诉决定书 | 同上 |
| 存疑不起诉 | 不起诉决定书 | 同上 |
| 退回补充侦查 | 退补提纲 | Word文档 |
| 认罪认罚 | 具结书+量刑建议书 | Word文档 |

### 3.9 认罪认罚具结书签署（如适用）

**签署流程**：
1. 确认嫌疑人自愿认罪认罚
2. 辩护人/值班律师在场
3. 向嫌疑人宣读具结书内容
4. 嫌疑人签字确认
5. 辩护人/值班律师签字确认
6. 全程同步录音录像
7. 未成年人：法定代理人到场并签字

## 阶段四：出庭支持公诉

### 4.1 庭前准备（开庭前3-5天）

**材料准备清单**：
```
□ 起诉书（纸质版+电子版）
□ 公诉意见书（书面稿）
□ 举证提纲（分组举证方案）
□ 质证提纲（逐项证据预判）
□ 答辩提纲（预判辩护观点并准备回应）
□ 讯问提纲（庭审讯问要点）
□ 相关法律条文打印件
□ 被告人基本信息表
□ 涉案财物清单
□ 多媒体示证材料（如有）
□ 类案检索材料（如有量刑争议）
```

### 4.2 庭前会议（如有）

**处理事项**：
- 管辖异议
- 回避申请
- 证据展示协商
- 非法证据排除申请
- 证人出庭申请
- 涉案财物处理

### 4.3 庭审操作

详见 `skills/通用能力/SKILL-阅卷与出庭技巧.md`。

**庭审流程**：
```
开庭
  ↓ 宣布案由、查明当事人到庭
法庭调查
  ↓ 公诉人宣读起诉书
  ↓ 被告人陈述
  ↓ 讯问被告人（按讯问提纲）
  ↓ 举证质证（按举证提纲分组举证）
法庭辩论
  ↓ 公诉人发表公诉意见
  ↓ 辩护人发表辩护意见
  ↓ 控辩双方互相辩论
最后陈述
  ↓ 被告人最后陈述
休庭/宣判
```

### 4.4 庭审后工作

- 核对庭审笔录
- 补充提交庭后材料（如有）
- 关注判决结果
- 准备判决审查

---

## 阶段五：判决审查

### 5.1 收到判决后

**审查要点**：
```
□ 判决认定的事实与起诉书指控是否一致
□ 罪名认定是否正确
□ 量刑是否适当（与量刑建议比较）
□ 有无遗漏的量刑情节
□ 程序是否合法
□ 被害人诉求是否得到回应
□ 涉案财物处置是否明确
```

### 5.2 处理决定

```
┌────────────────────────────────────┐
│           判决审查后处理             │
├──────────┬──────────┬──────────────┤
│ 认同判决  │ 不认同判决 │ 被害人申诉    │
│ 不抗诉    │ 考虑抗诉   │ 释法说理     │
│ 监督执行  │ 10日内决定 │ 引导申诉/自诉 │
└──────────┴──────────┴──────────────┘
```

## 阶段六：执行监督

### 6.1 判决生效后

- 确认判决是否生效（上诉期满/二审终审）
- 跟踪交付执行情况
- 监督刑罚执行合法性

### 6.2 减刑/假释/暂予监外执行审查

详见 `skills/刑事检察/SKILL-刑事执行监督.md`。

## 阶段七：归档结案

### 7.1 案卷归档

**归档清单**：
```
□ 所有法律文书（起诉书、判决书等）
□ 审查报告
□ 阅卷笔录
□ 讯问笔录
□ 证据清单
□ 量刑计算表
□ 出庭预案
□ 庭审笔录（复印件）
□ 判决审查意见
□ 办案流程记录
□ 认罪认罚具结书（如有）
□ 涉案财物处理材料
```

### 7.2 统一业务应用系统操作

- 案件流程信息录入
| 节点 | 操作 | 期限 |
|------|------|------|
| 受案登记 | 录入案件信息 | 当天 |
| 审查起诉 | 录入审查进度 | 持续 |
| 作出决定 | 录入处理决定 | 决定当天 |
| 移送法院 | 录入移送信息 | 移送当天 |
| 判决审查 | 录入审查意见 | 收到判决后 |
| 归档 | 完成电子归档 | 结案后 |

## 期限管控总表

| 环节 | 起算点 | 期限 | 可延长 | 注意事项 |
|------|--------|------|--------|---------|
| 审查批捕 | 收到提请 | 7日 | 不可延长 | 最后一天作出决定 |
| 审查起诉 | 受理当天 | 1个月 | +15日（重大复杂） | 退回补充侦查后重新计算 |
| 速裁审查起诉 | 受理当天 | 10日 | +5日（可能判1年以上） | |
| 退补侦查 | 移送补查 | 1个月 | 以2次为限 | 第2次退补后仍证据不足→存疑不起诉 |
| 自行补充侦查 | 决定自行补查 | 1个月 | 可多次 | 不受2次限制 |
| 一审（普通） | 受理 | 2个月 | +3个月（经批准） | 至迟3个月内宣判 |
| 一审（速裁） | 受理 | 10日 | +5日（判1年以上） | |
| 二审 | 受理 | 2个月 | +2个月（经批准） | |
| 抗诉期限 | 收到判决 | 10日 | 不可延长 | 不服裁定为5日 |

## 文书衔接关系图

```
案件受理
  │
  ├─→ 审查逮捕 ─→ 审查逮捕意见书 ─→ 批准/不批准逮捕决定书
  │
  ├─→ 审查起诉
  │     ├─→ 起诉书 ──────────→ 法院
  │     ├─→ 不起诉决定书 ────→ 送达当事人
  │     ├─→ 退补提纲 ────────→ 公安机关
  │     ├─→ 认罪认罚具结书 ──→ 附卷
  │     └─→ 量刑建议书 ──────→ 法院
  │
```
