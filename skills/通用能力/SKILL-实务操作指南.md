# 检察官助理实务操作指南

本指南解决检察官助理在日常工作中**真正会遇到的难点问题**，给出经过验证的最佳处理方案。

---

## 一、与公安机关的沟通与应对

### 1. 退补后公安消极补充侦查

**常见情形**：
- 退补提纲写得很清楚，公安补查敷衍了事
- 以"无法找到证人""客观原因无法取证"为由搪塞
- 补查材料与原移送材料完全重复
- 二次退补后仍然"查无实据"

**最佳应对方案**：

**退补时做到极致明确**：
```
❌ 错误写法："补充证人证言"
✅ 正确写法："调取被害人×××手机微信聊天记录（微信账号：×××），
           时间范围：2024年1月至2024年3月，
           目的：证明犯罪嫌疑人×××与被害人存在经济纠纷，
           即犯罪嫌疑人辩称的'借款'实为诈骗"
```

**补充侦查提纲应包含**：
1. 具体需要调取的证据名称和内容
2. 证据的存放位置/线索来源
3. 该证据要证明的具体事实
4. 调取的方法和途径
5. 不调取该证据的法律后果（"如无法补查，将因证据不足不起诉"）

**退补后跟踪**：
- 退补后第10天左右电话/当面询问进展
- 要求公安提供补查进度说明
- 必要时发《补充侦查催办函》

**二次退补仍不达标**：
- 拿着两次退补提纲和补查材料对比分析
- 在审查报告中详细列明"第一次退补未补查事项""第二次退补仍未补查事项"
- 作为存疑不起诉的充分理由

---

### 2. 公安移送案件材料不齐全

**常见情形**：
- 起诉意见书认定的犯罪事实在卷宗中找不到对应证据
- 关键证据缺失（如被害人陈述、鉴定意见未附卷）
- 被告人供述只有有罪供述，无辩解笔录
- 辨认笔录、现场勘验笔录不规范

**最佳应对方案**：

**接收案件时立即做清单核对**：
```
制作《案件材料核对清单》：
□ 起诉意见书
□ 立案决定书
□ 拘留证/逮捕证
□ 讯问笔录（×份，时间区间）
□ 被害人陈述（×份）
□ 证人证言（×份）
□ 物证及照片
□ 书证
□ 鉴定意见
□ 勘验/检查笔录
□ 辨认笔录
□ 视听资料/电子数据
□ 前科劣迹材料
□ 财产保全材料（如有）
□ 认罪认罚相关材料（如有）

缺失材料：
1. _______________
2. _______________
```

**处理方式**：
- **重要证据缺失**：直接发《补充材料函》，不进入审查程序
- **一般材料瑕疵**：记录在案，审查中发现问题再要求补正
- **程序性材料缺失**：如权利告知笔录缺失，可要求补正

### 3. 案件管辖争议

**常见情形**：
- 犯罪地和居住地法院都有管辖权，公安将案件移送至"好说话"的检察院
- 跨区域案件管辖不明
- 网络犯罪案件管辖争议

**最佳应对方案**：

**审查起诉阶段**：
- 发现管辖问题，层报检察长决定
- 需要移送管辖的，制作《移送管辖函》
- 不因管辖问题退回公安（除非管辖严重影响公正）

**提前预防**：
- 网络犯罪案件提前与公安沟通管辖依据
- 以犯罪结果发生地、被害人所在地为管辖连接点

---

### 4. 拘留延长的沟通

**常见情形**：
- 公安以"案情复杂"为由要求延长拘留期限
- 延长拘留期限的审批文书不规范

**最佳应对方案**：

**延长拘留期限审查要点**：
- 是否符合"流窜作案、多次作案、结伙作案"的条件
- 延长理由是否具体（不能笼统写"案情复杂"）
- 延长后的羁押期限是否仍合法

**不批准延长的应对**：
- 制作《不批准延长拘留期限说明书》
- 建议变更强制措施为取保候审或监视居住
- 在系统中做好记录

---

### 5. 立案监督中的公安不配合

**常见情形**：
- 公安对应当立案而不立案的线索消极处理
- 公安以"正在初查""证据不足"为由不立案
- 公安对立案监督通知不回复或敷衍回复

**最佳应对方案**：

**发出《通知立案书》后**：
- 15日内要求公安书面说明立案情况
- 未立案的，要求书面说明理由
- 理由不成立的，层报上级检察院

**跟踪落实**：
- 建立立案监督案件台账
- 每月跟踪一次进展
- 公安仍不立案的，可向同级人大常委会报告

## 二、与人民法院的沟通与应对

### 6. 法院不采纳量刑建议

**常见情形**：
- 法院认为量刑建议过重/过轻
- 法院以"案件具体情况"为由不采纳
- 认罪认罚案件法院不采纳量刑建议且未说明理由

**最佳应对方案**：

**庭审前预防**：
- 量刑建议充分说理，附详细计算过程
- 提前与法官沟通量刑建议（正式或非正式）
- 量刑建议留有适当弹性（幅度刑而非绝对确定刑）

**庭审中应对**：
- 法院当庭告知不采纳量刑建议的，应当记录在案
- 要求法院说明不采纳的理由
- 根据庭审情况调整量刑建议（有新的量刑情节时）

**判决后**：
- 仔细审查判决书的量刑说理
- 比较量刑建议与判决结果的差距
- 差距明显不合理的，考虑是否抗诉

**法律依据**（刑诉法第201条）：
- 法院一般应当采纳量刑建议
- 不采纳的，应当说明理由

---

### 7. 辩护人突然提出新证据或新观点

**常见情形**：
- 庭审中辩护人突然提交"新证据"
- 辩护人提出此前未阅卷发现的问题
- 被告人当庭翻供

**最佳应对方案**：

**庭前预防**：
- 庭前会议充分了解辩护人观点
- 提前准备辩护人可能提出的全部抗辩
- 仔细阅卷，不留死角

**庭审中应对**：

对于新证据：
- 审查证据是否属于"新证据"（是否在审查起诉阶段已存在）
- 要求休庭核实
- 必要时申请重新鉴定或补充调查

对于被告人翻供：
- 立即讯问翻供原因
- 宣读此前供述笔录
- 结合其他证据综合分析
- 说明供述自愿性和合法性

对于辩护观点：
- 当庭回应，围绕事实和证据展开
- 必要时申请休庭补充材料
- 不与辩护人进行情绪化辩论

---

### 8. 法院程序违法

**常见情形**：
- 未依法通知检察机关出庭
- 审判组织组成不合法
- 未依法保障辩护权
- 超期审理
- 违法缺席判决

**最佳应对方案**：

**庭审中发现**：
- 当庭提出口头纠正意见
- 要求书记员记录在案
- 严重违法的，建议休庭

**庭审后发现**：
- 制作《纠正审理违法意见书》
- 在法定期限内提出
- 跟踪法院纠正情况

**判决后发现**：
- 作为抗诉理由之一
- 在抗诉书中详细列明程序违法事实

---

### 9. 判决后被害人信访压力

**常见情形**：
- 被害人认为判决过轻，到检察院上访
- 被害人家属情绪激动，多次缠访
- 被害人通过网络媒体施压

**最佳应对方案**：

**接待规范**：
1. 有专门接待场所，两人以上接待
2. 耐心倾听，不推诿、不激化
3. 记录信访诉求
4. 明确告知法律程序和时限

**处理流程**：
```
第一步：倾听诉求，记录要点
第二步：解释判决理由（如判决已生效）
第三步：告知救济途径：
  - 可以申诉（控告申诉部门）
  - 可以申请抗诉（审判监督部门）
  - 可以提起自诉（针对不起诉案件）
第四步：在法定期限内书面答复
```

**特殊情形应对**：

| 情形 | 应对 |
|------|------|
| 被害人情绪激动 | 安抚为主，不激化矛盾 |
| 多次缠访 | 层报领导，启动释法说理程序 |
| 网络舆情 | 及时报告，配合宣传部门 |
| 要求追究被告人民事责任 | 引导另行提起民事诉讼 |
| 申请国家赔偿 | 引导依法申请 |

**关键原则**：
- 不承诺"一定改判"
- 不泄露案件秘密
- 不与被害人发生冲突
- 每次接待后做好记录存档

---

### 10. 二审出庭准备要点

**常见情形**：
- 被告人上诉但未提出具体理由
- 辩护人提出新的辩护意见
- 被害人请求检察院抗诉

**最佳应对方案**：

**出庭前准备**：
1. 仔细阅读上诉状和辩护词
2. 审查一审判决是否确有错误
3. 准备出庭意见（支持/反对一审判决）
4. 准备答辩提纲

**出庭意见的两种方向**：

**维持一审判决**：
- 一审判决认定事实清楚，证据确实充分
- 量刑适当
- 程序合法
- 上诉理由不能成立

**建议改判**（检察机关认为一审判决确有错误）：
- 详细说明一审判决的错误之处
- 提出具体的改判建议
- 提交相关证据和法律依据

## 三、与其他机关的协调

### 11. 监委移送案件的衔接

**常见情形**：
- 监委移送案件的证据标准与公安移送不同
- 监委调查期间的言词证据在审查起诉阶段需要重新固定
- 监委移送的案件往往涉案金额大、关系复杂

**最佳应对方案**：

**证据审查重点**：
- 留置期间讯问笔录的合法性（是否全程录音录像）
- 涉案财物的查封、扣押、冻结手续
- 调取的书证、电子数据的来源合法性

**程序衔接**：
- 收到移送材料后10日内决定是否受理
- 受理后依法讯问犯罪嫌疑人
- 需要补充侦查的，可以退回监委补充调查（以二次为限，每次一个月）
- 也可以自行补充侦查

**特殊注意事项**：
- 监委移送的案件一般不适用认罪认罚从宽（需审查是否符合条件）
- 留置措施自动解除，变更为逮捕或其他强制措施
- 注意案件涉及的政治敏感性

---

### 12. 鉴定机构沟通

**常见情形**：
- 鉴定机构以各种理由拖延出鉴定意见
- 鉴定意见不明确或存在歧义
- 需要补充鉴定或重新鉴定

**最佳应对方案**：

**委托鉴定时**：
- 明确鉴定事项和要求
- 提供完整的鉴定材料
- 约定合理的鉴定期限

**鉴定意见审查**：
- 鉴定机构和鉴定人是否具有资质
- 鉴定材料来源是否合法
- 鉴定方法是否科学
- 鉴定结论是否明确

**鉴定意见有问题时**：
- 发函要求鉴定机构作出书面说明
- 必要时补充鉴定
- 问题严重的，建议重新鉴定

## 四、实务中的常见难题

### 13. 认罪认罚后的翻供

**常见情形**：
- 审查起诉阶段签署具结书后，审判阶段翻供
- 辩护人做无罪辩护，被告人不再认罪
- 共犯之间互相推诿

**最佳应对方案**：

**预防措施**：
- 签署具结书时全程录音录像
- 确保被告人充分理解认罪认罚的含义
- 确保辩护人/值班律师已提供有效法律帮助
- 被告人书面确认"系自愿签署"

**翻供后的处理**：

```
第一步：审查翻供原因
  - 是否受到外部干扰（家属、同案犯等）
  - 是否对罪名或事实有误解
  - 是否受到不当诱导

第二步：评估翻供对案件的影响
  - 仅有被告人供述定案的：翻供后可能证据不足
  - 有其他证据印证的：翻供不影响定罪

第三步：应对策略
  情形A：翻供不影响定罪
    - 在庭审中宣读此前供述
    - 结合其他证据综合分析
    - 说明翻供不具有合理性
    
  情形B：翻供后证据可能不足
    - 补充其他证据
    - 强化间接证据体系
    - 必要时建议休庭补充调查
    
  情形C：确实因翻供导致证据不足
    - 依法作出存疑不起诉决定
    - 不因被害人压力勉强起诉
```

**抗辩翻供的要点**：
- 强调供述的自愿性（同步录音录像证明）
- 强调供述的稳定性（多次供述一致）
- 强调供述与其他证据的印证关系
- 揭示翻供的不合理性

### 14. 证据灭失的紧急处理

**常见情形**：
- 关键证人准备出国/失联
- 电子数据面临被删除风险
- 物证保管不善面临毁损
- 被害人病情严重可能无法作证

**最佳应对方案**：

**紧急固定证据**：
- 向公安机关发出《证据保全建议函》
- 必要时商请公安机关先行调取
- 电子数据：建议立即提取并制作哈希值
- 证人证言：优先安排询问

**法律依据**：
- 刑诉法第139条（扣押物证、书证）
- 刑诉法第144条（查询、冻结财产）

**程序要求**：
- 紧急情况下可以先行调取，后补办手续
- 调取过程应当全程录音录像
- 制作详细的证据提取笔录

### 15. 涉案财物处置

**常见情形**：
- 涉案财物种类多、价值认定难
- 被告人家属要求返还被扣押财物
- 涉案财物保管不善导致损毁
- 财物处置与判决不一致

**最佳应对方案**：

**审查起诉阶段**：
1. 核查扣押、冻结手续是否合法
2. 核实涉案财物清单与实际是否一致
3. 涉案财物价值需要鉴定的，及时委托鉴定
4. 与法院沟通涉案财物处置方案

**常见问题处理**：

| 问题 | 处理方案 |
|------|---------|
| 涉案财物价值争议 | 委托价格认定机构鉴定 |
| 家属要求返还 | 审查是否属于涉案财物，不属于的依法返还 |
| 财物保管不善 | 制作情况说明，建议相关部门承担保管责任 |
| 判决未涉及涉案财物 | 建议法院补充判决或另行处理 |

**涉案财物处置建议**：
- 起诉书中列明涉案财物情况
- 量刑建议时一并提出财产刑建议
- 建议法院在判决中明确涉案财物处置

---

### 16. 羁押必要性审查的最佳时机

**常见情形**：
- 犯罪嫌疑人被逮捕后一直羁押至审判
- 案件久拖不决，嫌疑人长期羁押
- 嫌疑人家庭出现特殊情况（家属重病、未成年子女无人照料）

**最佳应对方案**：

**主动审查时机**：

| 时机 | 审查内容 |
|------|---------|
| 逮捕后1个月 | 是否需要继续羁押 |
| 退回补充侦查前 | 是否可以变更强制措施 |
| 二次退补后 | 是否符合不起诉条件 |
| 审查起诉期限届满前 | 是否需要延长或变更 |
| 案件移送法院后 | 建议法院变更强制措施 |
| 每次换押时 | 重新评估羁押必要性 |

**可以变更强制措施的情形**：
1. 犯罪嫌疑人认罪认罚
2. 案件证据已固定，不存在灭失风险
3. 赔偿被害人并取得谅解
4. 羁押期限已接近可能判处的刑期
5. 嫌疑人出现严重疾病
6. 案件事实已查清，无需继续羁押

**变更强制措施建议**：
- 制作《羁押必要性审查报告》
- 向办案单位提出变更建议
- 跟踪落实情况

### 17. 被害人诉求与法律适用的冲突

**常见情形**：
- 被害人要求重判，但法律规定应当从轻
- 被害人要求赔偿，但被告人无赔偿能力
- 被害人不同意不起诉决定

**最佳应对方案**：

**诉求合理但法律有限制时**：
- 耐心释法说理
- 解释法律规定和量刑标准
- 引导被害人通过合法途径表达诉求

**被害人不同意不起诉**：
- 告知申诉权（7日内向上级检察院申诉）
- 告知自诉权（被害人有证据证明的轻微刑事案件）
- 做好释法说理工作

**赔偿问题**：
- 积极促成双方和解
- 被告人无赔偿能力的，如实告知被害人
- 可以建议被害人另行提起民事诉讼
- 对于符合司法救助条件的，引导申请司法救助

---

### 18. 涉黑涉恶案件的特殊处理

**常见情形**：
- 案件涉及人数多、事实多
- 需要与公安机关反复沟通核实
- 证据标准要求更高

**最佳应对方案**：

**审查要点**：
1. 黑社会性质组织的四个特征：
   - 组织特征（较稳定的犯罪组织）
   - 经济特征（有组织地获取经济利益）
   - 行为特征（以暴力、威胁等手段多次实施犯罪）
   - 危害性特征（严重破坏经济、社会生活秩序）

2. 恶势力的认定：
   - 三人以上
   - 经常纠集在一起
   - 以暴力、威胁或其他手段
   - 在一定区域或行业内多次实施违法犯罪活动

**实务建议**：
- 提前介入侦查，引导取证
- 制作详细的案件事实表格
- 分层分级处理各被告人的刑事责任
- 认罪认罚的适用需要更加审慎

## 五、时间管理与效率提升

### 19. 案件期限管理

**期限一览表**：

| 环节 | 期限 | 延长条件 |
|------|------|---------|
| 审查批捕 | 7日 | 无 |
| 审查起诉 | 1个月 | 重大复杂延长15日 |
| 速裁案件审查起诉 | 10日 | 可延长至15日 |
| 退补侦查 | 1个月 | 以二次为限 |
| 延长侦查羁押期限 | 2个月 | 特殊情况可再延长 |

**期限管理技巧**：
- 建立《案件期限台账》
- 期限届满前7天提醒
- 合理利用延长审限（有正当理由）
- 避免不必要的退补（节约时间）

### 20. 效率提升建议

**标准化**：
- 常用文书模板化
- 常见问题答复模板化
- 审查报告模板化

**工具使用**：
- 使用本 skill 的文书格式化脚本生成 Word 文档
- 使用法律检索 skill 快速查找法条
- 使用案件分析 skill 进行系统性分析

**时间分配**：
- 上午：集中处理需要思考的工作（案件审查、文书起草）
- 下午：处理沟通协调工作（阅卷、会见、会议）
- 每周固定时间处理系统性事务（期限检查、案件清理）

## 六、常见话术模板

### 与公安沟通

**退补时**：
> "这个案件有以下几个关键事实还需要进一步查证，具体是……请你们在一个月内补充侦查完毕。"

**催办时**：
> "上次退补的材料，现在已经过了X天，想了解一下进展。有几个比较关键的证据需要尽快固定，比如……"

### 与法院沟通

**量刑建议不被采纳时**：
> "我们提出的量刑建议是基于……的考虑。请问合议庭认为量刑建议需要调整的具体理由是什么？我们希望能充分沟通。"

**程序违法时**：
> "刚才庭审中存在……的情况，根据刑事诉讼法第……条的规定，我们提出口头纠正意见，请记录在案。"

### 与被害人沟通

**释法说理时**：
> "我们理解您的心情。根据法律规定，这个案件的具体情况是……法院的判决是综合考虑了……的因素。如果您对判决不服，可以依法向上级法院上诉。"

**赔偿问题时**：
> "我们已经多次与被告方沟通赔偿事宜。目前被告人的经济状况是……建议您可以通过以下途径维护权益：一是……二是……"

### 与辩护人沟通

**认罪认罚协商时**：
> "我们已经充分审查了案件事实和证据。根据被告人的犯罪事实、情节和认罪认罚表现，我们的量刑建议是……请问辩护人有什么意见？"

> "辩护人提出的意见我们已经记录。关于这一点，我们的回应是……有以下证据可以印证……"
