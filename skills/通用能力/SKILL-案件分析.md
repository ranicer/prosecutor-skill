# 案件分析 Skill

对刑事案件进行系统性分析，包括犯罪构成、证据体系、量刑情节等。

---

## 分析框架

### 一、犯罪构成分析

#### 四要件分析法

**1. 犯罪主体**
- 刑事责任年龄（第17条）
  - 完全刑事责任能力：已满16周岁
  - 相对刑事责任能力：已满14/12周岁
  - 限制刑事责任能力：不满18周岁、尚未完全丧失辨认或控制能力的精神病人
  - 完全无刑事责任能力：不满12周岁、不能辨认或控制自己行为的精神病人
- 特殊主体身份（国家工作人员、公司人员等）

**2. 主观方面**
- 犯罪故意（直接故意/间接故意）
- 犯罪过失（疏忽大意/过于自信）
- 犯罪目的和动机

**3. 犯罪客体**
- 一般客体
- 同类客体（按刑法分则章节判断）
- 直接客体（具体侵犯的法益）

**4. 犯罪客观方面**
- 危害行为（作为/不作为）
- 危害结果
- 因果关系
- 犯罪的时间、地点、方法

### 二、此罪与彼罪分析

常见易混淆罪名对比：

| 对比罪名 | 区分要点 |
|---------|---------|
| 盗窃罪 vs 抢夺罪 | 是否使用暴力/胁迫 |
| 抢夺罪 vs 抢劫罪 | 是否对人使用暴力 |
| 诈骗罪 vs 敲诈勒索罪 | 欺骗 vs 威胁 |
| 故意伤害罪 vs 故意杀人罪 | 主观故意内容 |
| 聚众斗殴罪 vs 故意伤害罪 | 是否有争霸动机 |
| 寻衅滋事罪 vs 故意伤害罪 | 是否随意殴打 |

### 三、一罪与数罪分析

- **想象竞合**：一行为触犯数罪名，从一重处
- **牵连犯**：手段行为与目的行为分别触犯不同罪名，从一重处
- **数罪并罚**：数个独立行为分别触犯不同罪名

## 证据分析

### 证据体系分析

| 证据类型 | 证明事项 | 证明力 |
|---------|---------|--------|
| 被告人供述 | 犯罪事实 | 直接证据，需补强 |
| 被害人陈述 | 被害经过 | 直接证据 |
| 证人证言 | 案件事实 | 间接/直接证据 |
| 物证 | 犯罪工具、赃物等 | 客观性强 |
| 书证 | 合同、账目等 | 客观性强 |
| 鉴定意见 | 伤情、价格等 | 专业性强 |
| 视听资料 | 犯罪过程 | 直观 |
| 电子数据 | 通讯、交易记录 | 客观性强 |

### 证据链完整性

完整的证据链应包括：
1. 犯罪主体证据
2. 犯罪主观方面证据
3. 犯罪客观方面证据（行为、结果、因果关系）
4. 量刑情节证据

## 量刑分析

### 法定量刑情节

**从重/加重情节**：
- 累犯（第65条）
- 特定犯罪加重处罚（如入户抢劫、持枪抢劫）

**从轻/减轻情节**：
- 自首（第67条）
- 立功（第68条）
- 坦白（第67条第三款）
- 从犯（第27条）
- 未遂（第23条）
- 中止（第24条）
- 未成年人（第17条）
- 防卫过当（第20条）
- 避险过当（第21条）

**免除处罚情节**：
- 犯罪中止没有造成损害（第24条）
- 预备犯（第22条）
- 自首且犯罪较轻（第67条）

### 酌定量刑情节

- 认罪认罚
- 赔偿损失
- 取得谅解
- 退赃退赔
- 初犯、偶犯
- 犯罪动机
- 犯罪手段
- 犯罪后果
- 社会影响

### 量刑建议步骤

1. 确定量刑起点（基本犯罪构成事实）
2. 确定基准刑（增加刑罚量）
3. 根据量刑情节调节基准刑
4. 综合考虑确定宣告刑/量刑建议

参考：`references/量刑指南/常见犯罪量刑指引.md`

## 输出格式

### 案件分析报告

```
一、案件基本情况
二、犯罪构成分析
  （一）犯罪主体
  （二）主观方面
  （三）犯罪客体
  （四）犯罪客观方面
三、此罪与彼罪分析（如有争议）
四、一罪与数罪分析（如涉及数罪）
五、证据分析
  （一）证据体系
  （二）证据链完整性
  （三）证据问题（如有）
六、量刑情节分析
  （一）法定情节
  （二）酌定情节
七、分析结论
```
