# 案件分析 Skill

对刑事案件进行系统性分析，包括犯罪构成、证据体系、量刑情节等。采用法律三段论推理框架，确保分析的逻辑严密性。

---

## 分析框架（五步法）

```
Step 1：犯罪构成分析（四要件）
    ↓
Step 2：法律三段论推理（定性分析）
    ↓
Step 3：此罪与彼罪 / 一罪与数罪分析
    ↓
Step 4：证据分析 + 量刑情节分析
    ↓
Step 5：类案检索参考 + 输出分析报告
```

> 详细的三段论推理方法见 `SKILL-法律三段论推理.md`
> 详细的类案检索方法见 `SKILL-类案检索与参考.md`
> 详细的证据矩阵和量刑计算见 `SKILL-证据整合与量刑辅助.md`

## 一、犯罪构成分析（四要件分析法）

**1. 犯罪主体**
- 刑事责任年龄（第17条）
  - 完全刑事责任能力：已满16周岁
  - 相对刑事责任能力：已满14/12周岁
  - 限制刑事责任能力：不满18周岁、尚未完全丧失辨认或控制能力的精神病人
  - 完全无刑事责任能力：不满12周岁、不能辨认或控制自己行为的精神病人
- 特殊主体身份（国家工作人员、公司人员等）

**2. 主观方面**
- 犯罪故意（直接故意/间接故意）
- 犯罪过失（疏忽大意/过于自信）
- 犯罪目的和动机

**3. 犯罪客体**
- 一般客体
- 同类客体（按刑法分则章节判断）
- 直接客体（具体侵犯的法益）

**4. 犯罪客观方面**
- 危害行为（作为/不作为）
- 危害结果
- 因果关系
- 犯罪的时间、地点、方法

### 二、此罪与彼罪分析

常见易混淆罪名对比：

| 对比罪名 | 区分要点 |
|---------|---------|
| 盗窃罪 vs 抢夺罪 | 是否使用暴力/胁迫 |
| 抢夺罪 vs 抢劫罪 | 是否对人使用暴力 |
| 诈骗罪 vs 敲诈勒索罪 | 欺骗 vs 威胁 |
| 故意伤害罪 vs 故意杀人罪 | 主观故意内容 |
| 聚众斗殴罪 vs 故意伤害罪 | 是否有争霸动机 |
| 寻衅滋事罪 vs 故意伤害罪 | 是否随意殴打 |

### 三、一罪与数罪分析

- **想象竞合**：一行为触犯数罪名，从一重处
- **牵连犯**：手段行为与目的行为分别触犯不同罪名，从一重处
- **数罪并罚**：数个独立行为分别触犯不同罪名

## 证据分析

### 证据体系分析

| 证据类型 | 证明事项 | 证明力 |
|---------|---------|--------|
| 被告人供述 | 犯罪事实 | 直接证据，需补强 |
| 被害人陈述 | 被害经过 | 直接证据 |
| 证人证言 | 案件事实 | 间接/直接证据 |
| 物证 | 犯罪工具、赃物等 | 客观性强 |
| 书证 | 合同、账目等 | 客观性强 |
| 鉴定意见 | 伤情、价格等 | 专业性强 |
| 视听资料 | 犯罪过程 | 直观 |
| 电子数据 | 通讯、交易记录 | 客观性强 |

### 证据链完整性

完整的证据链应包括：
1. 犯罪主体证据
2. 犯罪主观方面证据
3. 犯罪客观方面证据（行为、结果、因果关系）
4. 量刑情节证据

## 量刑分析

### 法定量刑情节

**从重/加重情节**：
- 累犯（第65条）
- 特定犯罪加重处罚（如入户抢劫、持枪抢劫）

**从轻/减轻情节**：
- 自首（第67条）
- 立功（第68条）
- 坦白（第67条第三款）
- 从犯（第27条）
- 未遂（第23条）
- 中止（第24条）
- 未成年人（第17条）
- 防卫过当（第20条）
- 避险过当（第21条）

**免除处罚情节**：
- 犯罪中止没有造成损害（第24条）
- 预备犯（第22条）
- 自首且犯罪较轻（第67条）

### 酌定量刑情节

- 认罪认罚
- 赔偿损失
- 取得谅解
- 退赃退赔
- 初犯、偶犯
- 犯罪动机
- 犯罪手段
- 犯罪后果
- 社会影响

### 量刑建议步骤

1. 确定量刑起点（基本犯罪构成事实）
2. 确定基准刑（增加刑罚量）
3. 根据量刑情节调节基准刑
4. 综合考虑确定宣告刑/量刑建议

参考：`references/量刑指南/常见犯罪量刑指引.md`

## 二、法律三段论推理（定性分析核心）

对每个涉嫌罪名进行三段论分析：

### 大前提（法律规范）

```
《刑法》第×××条第×款规定："……（法条原文）……"
构成要件：
  1. 主体要件：……
  2. 主观要件：……
  3. 客体要件：……
  4. 客观要件：……
```

### 小前提（事实涵摄）

```
╔══════════════════════════════════════════════════════════╗
║                法 律 涵 摄 对 照 表                       ║
╠════════════════╦══════════════════╦══════════════════════╣
║ 构成要件        ║ 法律要求          ║ 本案事实             ║
╠════════════════╬══════════════════╬══════════════════════╣
║ 犯罪主体        ║                  ║                      ║
║ 主观方面        ║                  ║                      ║
║ 犯罪客体        ║                  ║                      ║
║ 客观方面        ║                  ║                      ║
╠════════════════╬══════════════════╬══════════════════════╣
║ 涵摄结论        ║                  ║                      ║
╚════════════════╩══════════════════╩══════════════════════╝
```

### 结论

```
大前提：《刑法》第×××条规定了×××罪的构成要件。
小前提：本案中，×××（事实概述），符合/不符合×××罪的全部构成要件。
结论：×××的行为构成/不构成×××罪。
```

> 详细的三段论推理方法见 `SKILL-法律三段论推理.md`

## 三、类案检索参考

在案件分析完成后，进行类案检索以验证分析结论：

**检索时机**：
- 定性有争议时（此罪彼罪）
- 量刑建议需要论证时
- 辩护人可能提出无罪辩护时

**检索方法**：
- 最高检指导性案例检索
- 人民法院案例库检索（rmfyalk MCP）
- 裁判文书网检索

**输出**：类案检索报告 + 对本案的参考价值分析

> 详细的类案检索方法见 `SKILL-类案检索与参考.md`

## 四、协作关系

```
用户请求案件分析
    ↓
SKILL-案件分析（本文件）→ 犯罪构成 + 证据审查 + 量刑分析
    ↓
SKILL-法律三段论推理 → 定性分析（此罪彼罪/一罪数罪）
    ↓
SKILL-类案检索与参考 → 检索同类案件参考
    ↓
SKILL-证据整合与量刑辅助 → 复杂案件的证据矩阵 + 量刑计算
    ↓
SKILL-文书起草 → 输出 Word 格式文书
```
