# 证据整合与量刑辅助 Skill

针对复杂案件（多罪名、多被告人、证据量大），提供从证据梳理到量刑建议的完整辅助流程。

---

## 适用场景

- 多被告人共同犯罪案件
- 涉及多个罪名的案件
- 证据材料超过50页的复杂案件
- 需要精确计算量刑的认罪认罚案件
- 需要制作完整审查报告的案件

## 第三步：量刑情节自动提取与分类

### 法定量刑情节识别清单

根据案件事实和证据，逐项勾选：

```
□ 自首（第67条）：犯罪后自动投案，如实供述
□ 坦白（第67条第三款）：到案后如实供述
□ 立功（第68条）：揭发他人犯罪/提供重要线索
□ 重大立功（第68条）：揭发他人重大犯罪/提供重大线索
□ 从犯（第27条）：在共同犯罪中起次要/辅助作用
□ 胁从犯（第28条）：被胁迫参加犯罪
□ 未遂（第23条）：已着手实行但未得逞
□ 中止（第24条）：自动放弃/有效防止结果发生
□ 预备（第22条）：为犯罪准备工具、制造条件
□ 防卫过当（第20条）：正当防卫明显超过必要限度
□ 避险过当（第21条）：紧急避险超过必要限度造成损害
□ 未成年人（第17条）：不满18周岁
□ 75周岁以上（第17条之一）：故意犯罪可以从轻/过失应当从轻
□ 限制责任能力（第18条）：尚未完全丧失辨认/控制能力
□ 累犯（第65条）：前罪执行完毕后5年内再犯
□ 特别累犯（第66条）：危害国家安全/恐怖活动/黑社会性质组织犯罪累犯
□ 造成被害人重伤/死亡（酌定从重）
```

```
□ 认罪认罚：  阶段：□侦查 □审查起诉 □审判
□ 赔偿被害人损失：金额______ 全额/部分

□ 退赃退赔：金额______ 全额/部分
```

## 第四步：量刑计算

### 计算步骤

**Step 1：确定量刑起点**

根据基本犯罪构成事实，在对应法定刑幅度内确定量刑起点。

示例（盗窃罪数额较大）：
```
基本犯罪构成：盗窃金额8000元（数额较大，3年以下有期徒刑）
量刑起点：8个月-1年2个月有期徒刑
```

**Step 2：确定基准刑**

在量刑起点基础上，根据其他影响犯罪构成的犯罪事实增加刑罚量。

示例：
```
量刑起点：10个月
+ 盗窃金额超出起点部分（8000-3000=5000元，每5000元增加1-2个月）：+1个月
= 基准刑：11个月
```

**Step 3：量刑情节调节**

根据各量刑情节的调节比例，逐项调节基准刑。

示例：
```
基准刑：11个月

调节情节：
1. 累犯（+10%-40%）：+20% → +2.2个月
2. 认罪认罚-审查起诉阶段（-20%）：-20% → -2.2个月
3. 赔偿并取得谅解（-40%）：-40% → -4.4个月
4. 初犯（-10%）：-10% → -1.1个月

调节后：11 + 2.2 - 2.2 - 4.4 - 1.1 = 5.5个月
```

**Step 4：确定宣告刑/量刑建议**

综合全案情况，在调节后的刑期基础上适当调整。

```
宣告刑/量刑建议：有期徒刑6个月，可适用缓刑
```

## 多被告人共同犯罪处理

### 共犯地位分析表

| 被告人 | 参与犯罪事实 | 行为方式 | 所起作用 | 地位认定 | 量刑建议 |
|--------|------------|---------|---------|---------|---------|
| 张三 | 事实1、2、3 | 组织策划 | 主要作用 | 主犯 | |
| 李四 | 事实1、2 | 直接实施 | 次要作用 | 从犯 | |
| 王五 | 事实1 | 提供帮助 | 辅助作用 | 从犯 | |

### 分别量刑原则

1. 各被告人的犯罪事实分别评价
2. 各被告人的量刑情节分别适用
3. 共同犯罪中的地位和作用作为各自量刑情节
4. 主犯按照其组织、指挥的全部犯罪处罚
5. 从犯应当从轻、减轻或免除处罚

## 多罪名案件处理

### 数罪并罚计算

```
罪名1：有期徒刑___年___月
罪名2：有期徒刑___年___月
总和刑期：___年___月
数刑中最高刑期：___年___月

决定执行刑期：
  总和刑期不满35年的，最高不超过20年
  总和刑期在35年以上的，最高不超过25年

建议执行：有期徒刑___年___月
```

### 一罪与数罪分析要点

| 情形 | 处理方式 |
|------|---------|
| 想象竞合 | 从一重处（择一重罪处罚） |
| 牵连犯 | 从一重处（手段与目的分别触犯不同罪名） |
| 连续犯 | 以一罪论处（连续实施同种犯罪） |
| 同种数罪 | 一般不并罚（判决宣告前发现的同种数罪） |
| 异种数罪 | 数罪并罚 |

## 使用示例

```
用户：这个案件比较复杂，3个被告人，涉及盗窃和掩饰隐瞒两个罪名，
      证据材料有80多页，帮我梳理证据并计算量刑。

助手：[启动证据整合与量刑辅助skill]
1. 结构化提取案件信息（被告人信息、犯罪事实、证据清单）
2. 构建证据矩阵（各证据×各待证事实）
3. 逐项识别量刑情节（从重/从轻/减轻）
4. 分别计算各被告人各罪名的量刑
5. 数罪并罚计算
6. 生成量刑计算表
7. 输出起诉书（Word格式）
```
