# 法律检索 Skill

进行系统性的法律检索，覆盖法律、行政法规、司法解释、部门规章、地方性法规和指导性案例。

---

## 触发条件

- "查找法条"
- "法律规定是什么"
- "有什么司法解释"
- "找相关案例"
- "指导性案例"
- "类似案件怎么判"

---

## 检索流程

### 第一步：明确检索目标
- 确认涉及的法律问题
- 确认案件类型（刑事/民事/行政/公益诉讼）
- 确认检索范围（法律/司法解释/案例）

### 第二步：按效力层级检索

**第一层：法律**
- 全国人大及其常委会制定的法律
- 如：刑法、刑事诉讼法、民法典等

**第二层：行政法规**
- 国务院制定的行政法规

**第三层：司法解释**
- 最高人民法院、最高人民检察院联合或单独发布的司法解释
- 司法解释性质文件

**第四层：部门规章**
- 国务院各部委制定的规章

**第五层：地方性法规**
- 省、自治区、直辖市及设区的市人大及其常委会制定

### 第三步：检索数据库

| 数据库 | 网址 | 说明 |
|--------|------|------|
| 国家法律法规数据库 | flk.npc.gov.cn | 官方权威 |
| 北大法宝 | pkulaw.cn | 全面专业 |
| 法信 | faxin.cn | 案例丰富 |
| 人民法院案例库 | rmfyalk.court.gov.cn | 指导性案例 |
| 最高人民法院官网 | court.gov.cn | 司法文件 |
| 最高人民检察院官网 | spp.gov.cn | 检察案例 |

### 第四步：输出检索报告

```
法律检索报告

一、检索问题
二、检索过程
  检索关键词：
  检索数据库：
  检索时间：
三、检索结果
  （一）相关法律
  （二）相关司法解释
  （三）相关案例
四、法律分析
五、结论与建议
```

## MCP 服务器检索

如已配置 MCP 服务器，可直接调用：

```
# 国家法律法规数据库 MCP
搜索法律法规：flk_npc_search(query="关键词")

# 人民法院案例库 MCP  
搜索指导案例：rmfyalk_search(query="关键词")
```

---

## 注意事项

1. **时效性**：核实法律法规是否为最新有效版本
2. **地域性**：注意地方性法规的适用范围
3. **上位法优先**：下位法与上位法冲突的，适用上位法
4. **新法优于旧法**：同一效力层级，新规定优先
5. **特别法优于一般法**：特别规定优先于一般规定
