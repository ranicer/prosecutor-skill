# 法律检索 Skill

进行系统性的法律检索，覆盖法律、行政法规、司法解释、部门规章、地方性法规和指导性案例。结合 RAG（检索增强生成）方法论，提供精准的法律信息检索。

---

## 触发条件

- "查找法条"
- "法律规定是什么"
- "有什么司法解释"
- "找相关案例"
- "指导性案例"
- "类似案件怎么判"
- "帮我查一下×××法条"

---

## 一、标准检索流程

```
用户法律问题
    ↓
Step 1：明确检索目标
    ↓
Step 2：提取关键词（罪名/法条/事实要素）
    ↓
Step 3：按效力层级逐层检索
    ↓
Step 4：关联法条检索（扩大检索范围）
    ↓
Step 5：输出检索报告
```

### 第一步：明确检索目标

- 确认涉及的法律问题
- 确认案件类型（刑事/民事/行政/公益诉讼）
- 确认检索范围（法律/司法解释/案例）

### 第二步：关键词提取

| 案件信息 | 检索关键词 |
|---------|----------|
| 涉嫌罪名 | 罪名全称 + 分则章节 |
| 犯罪手段 | 手段类型 + 相关司法解释 |
| 特殊情节 | "自首""累犯""认罪认罚"等 |
| 争议焦点 | 此罪彼罪对比关键词 |

### 第三步：按效力层级检索

**第一层：法律**
- 全国人大及其常委会制定的法律
- 如：刑法、刑事诉讼法、民法典等
- 优先从 `references/核心法律/` 本地数据库检索

**第二层：行政法规**
- 国务院制定的行政法规

**第三层：司法解释**
- 最高人民法院、最高人民检察院联合或单独发布的司法解释
- 司法解释性质文件
- 重点关注"两高"关于具体罪名的解释

**第四层：部门规章**
- 国务院各部委制定的规章

**第五层：地方性法规**
- 省、自治区、直辖市及设区的市人大及其常委会制定

### 第四步：关联法条检索

找到核心法条后，还需检索关联法条：
- 程序法依据（刑诉法相关条文）
- 量刑相关条文（刑法总则中关于该罪名的量刑规定）
- 司法解释中的具体适用规则

### 第五步：输出检索报告

```
法律检索报告

一、检索问题
二、检索过程
  检索关键词：
  检索数据库：
  检索时间：
三、检索结果
  （一）相关法律
  （二）相关司法解释
  （三）相关案例
四、法律分析
五、结论与建议
```

## 二、MCP 服务器实操指引

### 配置方法

在 Claude Code 的 MCP 配置中添加以下服务器：

```json
{
  "flk-npc": {
    "type": "http",
    "url": "http://localhost:18062/mcp"
  },
  "rmfyalk": {
    "type": "http",
    "url": "http://localhost:18061/mcp"
  }
}
```

### flk-npc：国家法律法规数据库

**功能**：检索法律法规原文（无需认证）

**使用场景**：
- 查找法律条文原文
- 确认法条最新有效版本
- 检索行政法规、部门规章

**调用方式**：
```
搜索法律条文：flk_npc_search(query="盗窃罪 数额较大")
查看法条详情：flk_npc_get(article_id="xxx")
```

### rmfyalk：人民法院案例库

**功能**：检索指导性案例和参考案例（需 Token 认证）

**使用场景**：
- 检索同类案件的裁判结果
- 查找指导性案例的裁判要旨
- 获取案例中的法律适用分析

**调用方式**：
```
搜索案例：rmfyalk_search(query="盗窃罪 认罪认罚 缓刑")
查看案例详情：rmfyalk_get(case_id="xxx")
```

### RAG 检索增强流程

```
用户问题 → 关键词提取 → MCP 法条检索 → 法条原文获取
                                              ↓
用户问题 → 关键词提取 → MCP 案例检索 → 案例详情获取
                                              ↓
                                    法条 + 案件事实 → 生成分析报告
```

**RAG 优势**：
- 法条引用准确（直接来自官方数据库）
- 案例参考真实（来自人民法院案例库）
- 分析基于最新法律规定

## 五、注意事项

1. **时效性**：核实法律法规是否为最新有效版本
2. **地域性**：注意地方性法规的适用范围
3. **上位法优先**：下位法与上位法冲突的，适用上位法
4. **新法优于旧法**：同一效力层级，新规定优先
5. **特别法优于一般法**：特别规定优先于一般规定
6. **司法解释引用**：引用司法解释时注明文号（如"法释〔20××〕×号"）
