# 类案检索与参考 Skill

为检察官助理提供系统化的类案检索方法，辅助在案件审查、量刑建议、出庭支持公诉等环节找到具有参考价值的类似案例。

---

## 一、类案检索的时机

在以下场景应当主动进行类案检索：

| 场景 | 检索目的 |
|------|---------|
| 定性争议（此罪彼罪） | 检索同类案件的定性做法 |
| 量刑建议有分歧 | 检索同类案件的量刑区间 |
| 辩护人提出无罪辩护 | 检索同类案件的有罪判例 |
| 认罪认罚量刑协商 | 检索同类案件的从宽幅度 |
| 不起诉决定 | 检索同类案件的不起诉先例 |
| 出庭准备 | 准备类案检索报告应对辩护观点 |
| 业务竞赛/论辩赛 | 积累典型案例 |

## 二、检索来源与方法

### （一）最高人民检察院指导性案例

**检索路径**：最高检官网 → 指导性案例

**案例效力**：各级检察院办理类似案件应当参照

**检索方法**：
1. 按罪名关键词检索
2. 按案例编号检索（如"检例第×××号"）
3. 按业务领域检索（刑事/民事/行政/公益诉讼）

**重点关注**：
- 指导性案例的"要旨"部分（明确法律适用规则）
- 参照要点（如何在类似案件中适用）

### （二）人民法院案例库

**检索路径**：rmfyalk.court.gov.cn（需Token认证）

**MCP 服务器配置**：
```json
{
  "rmfyalk": {
    "type": "http",
    "url": "http://localhost:18061/mcp"
  }
}
```

**检索方法**：
1. 按案由检索（如"盗窃罪""故意伤害罪"）
2. 按关键词检索（犯罪手段、数额、情节等）
3. 按裁判结果检索（量刑区间）

### （三）国家法律法规数据库（法条关联检索）

**检索路径**：flk.npc.gov.cn（无需认证）

**MCP 服务器配置**：
```json
{
  "flk-npc": {
    "type": "http",
    "url": "http://localhost:18062/mcp"
  }
}
```

**检索方法**：通过法条反查适用该法条的案例

### （四）中国裁判文书网

**检索路径**：wenshu.court.gov.cn

**检索方法**：
1. 按法院层级筛选（基层/中级/高级/最高）
2. 按地域筛选（同辖区参考价值更高）
3. 按时间筛选（近两年案例参考价值更高）
4. 按审判程序筛选（一审/二审/再审）

### （五）最高检公报案例 / 典型案例

定期发布的典型案例，对检察实务有直接参考价值

## 四、类案在检察工作中的运用

### （一）量刑建议中的运用

**操作步骤**：
1. 检索同类案件的量刑结果
2. 统计量刑区间（最低刑期、最高刑期、中位数）
3. 分析本案与参考案例的差异
4. 论证量刑建议的合理性

**量刑参考区间表**：
```
类案检索结果汇总：
- 检索案件数量：×件
- 量刑区间：×个月 至 ×年×个月
- 中位数：×年×个月
- 认罪认罚案件从宽幅度：×% 至 ×%
- 缓刑适用率：×%
```

### （二）出庭中的运用

- 准备《类案检索报告》作为庭审参考
- 回应辩护人"量刑过重"时引用同类案件量刑
- 回应"此罪彼罪"争议时引用定性先例

### （三）审查起诉中的运用

- 定性有争议时，参考同类案件的定性做法
- 决定是否起诉时，参考同类案件的处理结果
- 认罪认罚量刑协商时，参考同类案件的从宽幅度

## 五、检索注意事项

1. **优先效力层级**：指导性案例 > 公报案例 > 典型案例 > 参考性案例
2. **同辖区优先**：本省/本市案例参考价值更高
3. **时效性**：优先检索近两年案例，注意法律修改的影响
4. **差异分析**：找到参考案例后必须分析本案与参考案例的差异点
5. **全面检索**：不能只检索支持自己观点的案例，也要检索相反观点的案例
6. **保密义务**：检索到的案件信息不得泄露
