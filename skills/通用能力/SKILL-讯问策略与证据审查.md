# 讯问策略与证据审查

> 本模块系统梳理检察官在讯问、认罪认罚协商、供述审查中的核心技巧与方法论。内容整合自一线办案实践和审讯心理学研究成果，聚焦法律框架内的实务运用，兼顾"攻"（讯问突破）与"防"（合法性审查）双线能力。

---

## 一、讯问心理学基础

### 1.1 嫌疑人供述的心理动机

理解嫌疑人为什么开口，是所有讯问策略的起点：

| 动机类型 | 心理机制 | 讯问中的利用方式 |
|---------|---------|-----------------|
| **趋利避害** | 本能地追求最大利益、回避最坏结果 | 充分展示认罪从宽的实利，让不认罪的后果清晰可见 |
| **侥幸心理** | 相信"他们没有证据""咬死不认就没法定罪" | 逐步出示证据，瓦解"证据不足"的幻想 |
| **心理防线崩溃** | 长期对抗带来的精神疲惫、孤立无援感 | 耐心施加心理压力，等待临界点 |
| **良心发现** | 道德内疚感驱动 | 唤起对被害人的共情、对家人的牵挂 |
| **利益交换** | 用供述换取从宽、保护他人 | 明确告知认罪认罚的具体从宽幅度 |
| **解脱心理** | 长期隐瞒带来的精神重压 | 营造"说出来就轻松了"的氛围 |

**实务要点**：多数嫌疑人的供述动机是混合型的。讯问前先评估嫌疑人最可能被哪种动机驱动，再选择对应的突破策略。

### 1.2 常见对抗心理及突破方向

| 对抗类型 | 典型表现 | 突破方向 |
|---------|---------|---------|
| **硬扛型** | 沉默不语、拒绝回答 | 证据施压 + 时间压力 + 利益引导 |
| **狡辩型** | 编造理由、推卸责任 | 矛盾揭露 + 细节追问 + 证据对比 |
| **翻供型** | 先供后翻、反复变化 | 稳定性追问 + 供述印证 + 排除外部干扰 |
| **讨价还价型** | 承认部分事实、否认关键事实 | 分离事实与定性 + 认罪认罚引导 |
| **卖惨型** | 强调家庭困难、身体状况 | 人文关怀 + 同情但不承诺 + 回归案件事实 |

## 二、合法讯问策略体系

### 2.1 信息控制策略

**核心原理**：利用嫌疑人对"检察院掌握了多少证据"的不确定感，制造心理压力。

#### A. 选择性信息透露

- **操作方法**：不一次性出示全部证据，而是有节奏地逐步释放
- **心理效果**：嫌疑人会认为"他们还有更多我不知道的证据"
- **合法性边界**：必须基于真实存在的证据，不得虚构或夸大
- **话术示例**：
  > "你的情况我们已经掌握了，你自己说和我们替你说，性质是不一样的。"

#### B. 同案犯信息利用

- **操作方法**：告知嫌疑人同案犯已经供述（在确实有此供述的前提下）
- **心理效果**：制造"囚徒困境"——不说就吃亏
- **合法性边界**：必须基于真实供述；不得伪造同案犯供述
- **话术示例**：
  > "你的同案×××已经把事情经过讲清楚了。你现在如实交代，还能争取从宽处理。如果你和他说的不一致，那谁在说谎，我们一查就清楚。"

#### C. 客观证据暗示

- **操作方法**：暗示已掌握电子数据、监控、DNA等客观证据
- **心理效果**：让嫌疑人认识到"抵赖没有意义"
- **合法性边界**：必须确实掌握了相关证据
- **话术示例**：
  > "现在技术手段很先进，电子数据不会说谎。你手机里的聊天记录、转账记录，我们都有。"

### 2.2 心理突破策略

#### A. 矛盾揭露法（最核心、最合法的讯问技巧）

- **操作方法**：逐条指出嫌疑人供述中的矛盾之处，迫使其面对事实
- **心理效果**：供述的虚假性被揭露，心理防线动摇
- **完全合法**，是讯问的基本功
- **话术示例**：
  > "你说案发当天你在家，但监控显示你当晚8点出现在×××。你说你没有和被害人联系过，但被害人手机里有你发的17条微信。你怎么解释这些矛盾？"

#### B. 情感突破法

- **操作方法**：从嫌疑人的家庭、子女、父母等情感软肋切入
- **心理效果**：唤起愧疚感和责任感，降低对抗意愿
- **合法性边界**：不得以家属安全相威胁；可以正常提及家人唤起情感
- **话术示例**：
  > "你孩子今年上小学了吧？你希望他以后怎么看你这件事？你现在如实交代，争取从宽，早点回去陪孩子，这才是负责任的做法。"

#### C. 台阶法（给嫌疑人一个"体面认罪"的理由）

- **操作方法**：帮嫌疑人找到一个"可以接受"的认罪理由，降低认罪的心理门槛
- **心理效果**：嫌疑人觉得认罪不是"认输"，而是"理性选择"
- **完全合法**，是认罪认罚协商中的核心技巧
- **话术示例**：
  > "我知道你不是坏人，这件事确实是一时冲动。法律上叫'激情犯罪'，和预谋犯罪的性质是不一样的。你现在如实交代，加上认罪认罚从宽，量刑上会有明显的优惠。"

#### D. 证据突击法

- **操作方法**：在嫌疑人以为检察院没有关键证据时，突然出示
- **心理效果**：瞬间击溃侥幸心理
- **完全合法**
- **操作要点**：
  1. 讯问前充分评估哪些证据对嫌疑人冲击最大
  2. 先让嫌疑人充分表达其"无罪"版本
  3. 在其供述固定后再出示关键证据
  4. 观察其心理反应，趁势追问

#### E. 沉默对视法

- **操作方法**：在关键问题上不追问，而是沉默注视嫌疑人
- **心理效果**：沉默本身制造的心理压力远大于追问
- **合法性**：基本合法，但持续时间不宜过长（建议不超过3-5分钟）
- **适用场景**：嫌疑人明显在撒谎或犹豫时

### 2.3 时间与节奏控制

- **提审节奏**：不宜过于密集（造成疲劳），也不宜间隔过长（给嫌疑人喘息机会）
- **讯问时间**：上午精神状态好时讯问效果更佳；避免深夜疲劳讯问
- **单次时长**：一般控制在2-4小时为宜，关键突破时刻可适当延长，但需保障饮食休息
- **暂停策略**：感觉嫌疑人即将突破但还在犹豫时，可以暂停讯问，让其自己消化思考

### 2.4 认罪认罚协商中的心理策略

#### 协商前准备

1. **心理画像**：根据案卷材料预判嫌疑人的性格特点、心理状态
2. **底线设定**：明确己方的量刑建议底线（内心确认，不轻易亮出）
3. **筹码盘点**：列明所有可以用于交换的从宽情节（退赃、谅解、自首等）
4. **辩护人预判**：提前分析辩护人可能提出的量刑意见

#### 协商中的策略运用

| 策略 | 具体做法 | 心理原理 |
|------|---------|---------|
| **先声夺人** | 开场先展示证据优势和法律依据 | 建立心理优势地位 |
| **留有余地** | 量刑建议适当留出空间 | 给对方"争取"的满足感 |
| **利益交换** | 用从宽换取退赃、谅解等 | 趋利避害心理 |
| **及时锁定** | 达成一致后立即签署具结书 | 防止夜长梦多 |
| **情理结合** | 表达对嫌疑人处境的理解 | 降低对抗心理 |
| **最后通牒** | 告知协商窗口期有限 | 时间压力促决策 |

#### 协商话术示例

**开场**：
> "×××，案件事实和证据我们已经审查清楚了。今天找你谈，是因为认罪认罚从宽制度给了你一个争取从轻处理的机会。我先把法律规定和案件情况跟你说清楚。"

**展示证据优势**：
> "本案有以下证据：第一……第二……第三……这些证据形成了完整的证据链。我建议你认真考虑一下。"

**引导认罪**：
> "如果你自愿认罪认罚，根据法律规定和量刑指导意见，可以判处×××。如果不认罪，根据全案证据，可能判处×××。差额是很明显的。"

**应对犹豫**：
> "我理解你需要考虑。但提醒你，认罪认罚的从宽幅度是有条件的，如果你的态度发生变化，量刑建议也会相应调整。"

**应对讨价还价**：
> "你的意见我记录了。关于×××这一点，我们的考虑是×××。如果被告人能够退赃退赔/取得被害人谅解，我们可以适当调整。你看怎么样？"

## 三、供述审查方法

### 3.1 供述稳定性审查

**审查要点**：

| 审查维度 | 具体内容 | 关注点 |
|---------|---------|--------|
| **多次供述对比** | 比对历次讯问笔录的异同 | 核心事实是否一致、细节是否吻合 |
| **供述变化节点** | 分析供述何时开始变化 | 变化是否在特定事件后（如律师介入、同案犯供述变化） |
| **变化原因** | 追问供述变化的原因 | 是否合理、是否受到外部影响 |
| **供述自然性** | 供述是否像"自己说的" | 过于整齐划一的供述可能被"教"过 |

**异常信号识别**：
- 多次供述高度雷同（几乎像复制粘贴）→ 可能存在笔录制作不规范
- 供述只讲有罪部分，没有辩解内容 → 不自然，真实供述通常有辩解
- 关键细节在多次供述中完全一致（连用词都一样）→ 需核查同步录音录像
- 供述与客观证据完美吻合，没有任何偏差 → 需审查是否为"先有结论再找口供"

### 3.2 供述自愿性审查

**审查方法**：

1. **讯问嫌疑人**：
   > "你在侦查阶段的供述是否属实？是否有被强迫、威胁、引诱的情况？"
   > "你当时为什么作出有罪供述？"

2. **审查同步录音录像**：
   - 录音录像是否完整（有无剪辑、中断）
   - 讯问过程中有无不当言行
   - 嫌疑人精神状态是否正常
   - 权利告知是否到位

3. **审查讯问笔录**：
   - 讯问时间是否合规（是否超时、是否有休息间隔）
   - 讯问地点是否合规
   - 讯问人数是否合规（二人以上）
   - 权利告知是否完整

4. **审查身体状况**：
   - 入所体检记录
   - 讯问前后的身体状况对比
   - 有无伤痕记录

### 3.3 供述与客观证据印证审查

**审查框架**：

```
供述内容 → 与客观证据逐项比对 → 判断印证程度

印证程度分类：
A. 完全印证：供述与所有客观证据吻合 → 可信度高
B. 基本印证：供述与主要证据吻合，个别细节有出入 → 需进一步核实
C. 部分印证：供述与部分证据吻合，关键事实有矛盾 → 需重点审查
D. 不印证：供述与客观证据矛盾 → 高度可疑，需排除非法取证可能
```

**关键审查点**：
- 供述中的"隐蔽性细节"是否与客观事实吻合（只有作案人才能知道的细节）
- 供述的时间线是否与客观证据时间线一致
- 供述的犯罪动机、手段是否与客观证据反映的情况吻合

## 四、非法证据识别与应对

### 4.1 非法取证行为的类型

| 类型 | 具体表现 | 法律后果 |
|------|---------|---------|
| **暴力取证** | 殴打、捆绑、违法使用戒具 | 供述应当排除 |
| **变相暴力** | 冻、饿、晒、烤、疲劳审讯 | 供述应当排除 |
| **威胁取证** | 以暴力或损害合法权益相威胁 | 供述应当排除 |
| **引诱取证** | 以虚假从轻承诺引诱供述 | 供述应当排除 |
| **欺骗取证** | 以虚假证据、虚假同案犯供述欺骗 | 供述应当排除 |
| **疲劳审讯** | 连续讯问超过12小时未保障休息 | 供述可能被排除 |
| **程序违法** | 未告知权利、单人讯问、异地讯问未合规 | 供述可能被排除 |

### 4.2 "合法策略"与"非法取证"的界限

| 做法 | 合法性判断 | 依据 |
|------|-----------|------|
| 告知认罪认罚从宽政策 | 合法 | 刑事诉讼法第15条 |
| 出示真实掌握的证据 | 合法 | 讯问策略 |
| 告知同案犯已供述 | 合法（需真实） | 信息控制策略 |
| 承诺"认了就放你" | **非法**（虚假承诺） | 刑事诉讼法第56条 |
| 威胁"不认就重判" | **非法**（威胁） | 刑事诉讼法第56条 |
| 伪造证据施压 | **非法**（欺骗） | 刑事诉讼法第56条 |
| 沉默对视（3-5分钟） | 合法 | 讯问策略 |
| 连续讯问超12小时 | **非法**（疲劳审讯） | 两高三部规定 |
| 深夜讯问（凌晨2-6点） | 灰色地带 | 需个案判断 |
| 以家属情况唤起情感 | 合法（正常范围） | 情感突破法 |
| 以家属安全相威胁 | **非法**（威胁） | 刑事诉讼法第56条 |

#### 常见诱供话术识别（实务归纳）

以下为讯问中常见的、可能构成诱供的话术模式，检察官在审查讯问合法性时需重点识别：

| 话术类型 | 典型表述 | 风险点 |
|---------|---------|--------|
| **虚假从轻承诺** | "你认了就判缓刑""签了就放你走" | 承诺的从轻幅度超出法律规定 |
| **虚假证据暗示** | "我们已经找到你的指纹了"（实际没有） | 伪造证据，构成欺骗 |
| **虚假同案犯供述** | "你同伙已经全说了"（实际没人说） | 伪造供述，构成欺骗 |
| **威胁性暗示** | "不配合的话，你老婆也会受牵连" | 以家属相威胁 |
| **利益交换诱导** | "你配合的话，我们可以帮你取保" | 超出法定权限的承诺 |
| **选择性告知** | 只告知不利证据，隐瞒有利证据 | 可能影响供述自愿性 |

**审查要点**：讯问笔录中如果嫌疑人的供述与客观证据高度吻合，但讯问过程存在上述话术，需重点核查同步录音录像确认供述自愿性。

### 4.3 非法证据排除的审查流程

```
嫌疑人提出被非法取证
  ↓
初步审查：讯问笔录、体检记录、同步录音录像
  ↓
┌─ 明显非法 → 启动排除程序 → 排除供述
│
├─ 疑似非法 → 要求侦查机关说明 → 核查
│     ├─ 核实为非法 → 排除
│     └─ 核实合法 → 不排除
│
└─ 不存在非法取证情形 → 不排除
```

**审查重点**：
1. 同步录音录像是否完整连续
2. 讯问时间、地点、人员是否合规
3. 嫌疑人身体状况是否正常
4. 供述内容是否自然（非"背诵式"供述）
5. 多次供述之间的变化是否合理

## 五、翻供的应对策略

### 5.1 翻供原因分析

| 原因类型 | 具体表现 | 应对方向 |
|---------|---------|---------|
| **外部干扰** | 家属、同案犯、律师教唆 | 排除干扰源 + 强化证据体系 |
| **侥幸心理** | 认为翻供可能影响定罪 | 出示客观证据 + 说明翻供后果 |
| **供述非自愿** | 侦查阶段确实存在不当讯问 | 审查取证合法性 + 排除非法证据 |
| **事实认知偏差** | 对行为性质有不同理解 | 释法说理 + 法律适用分析 |
| **量刑博弈** | 试图通过翻供获取更有利的量刑 | 说明认罪认罚从宽 + 翻供后果 |

### 5.2 翻供后的处理流程

```
嫌疑人翻供
  ↓
第一步：确认翻供内容
  > "你说之前的供述不属实，具体哪些部分不属实？你现在的说法是什么？"
  ↓
第二步：分析翻供原因
  > "你之前为什么那样说？现在为什么改变了？是什么原因？"
  ↓
第三步：对比多次供述
  - 列明历次供述一致的部分
  - 指出翻供内容与此前供述的矛盾
  ↓
第四步：结合客观证据
  - 即使排除供述，其他证据能否定罪
  - 供述中的"隐蔽性细节"是否有客观证据印证
  ↓
第五步：评估翻供对案件的影响
  - 影响定罪 → 补强客观证据
  - 不影响定罪 → 庭审中应对
  - 确因翻供导致证据不足 → 依法处理
```

### 5.3 庭审中应对翻供的技巧

**五步法**：

**第一步：稳住阵脚**
- 不要表现出惊讶或慌张
- 翻供不等于败诉，关键看证据体系

**第二步：确认翻供内容**
> "被告人×××，你刚才说侦查阶段的供述不是事实。请你明确一下，具体在哪些关键事实上不一致？"

**第三步：对比分析**
> "根据你在×月×日、×月×日、×月×日的三次供述，你均稳定供述了以下事实：第一……第二……这三次供述的主要内容是一致的。你今天说不属实，请问之前三次为什么都不属实？"

**第四步：结合其他证据**
> "即使抛开你的供述，本案还有以下证据能够印证：第一，证人×××的证言……第二，书证……第三，物证……这些证据形成了完整的证据链条。"

**第五步：固定翻供原因**
> "请你说明翻供的具体原因。是有人让你这样说的吗？是你自己的想法？"
- 这一步非常关键，可以揭示翻供是否受到不当影响

## 六、审查起诉阶段的讯问要点

### 6.1 首次讯问的核心任务

1. **核实身份**：确认犯罪嫌疑人身份信息
2. **权利告知**：委托辩护人、申请法律援助、认罪认罚从宽
3. **听取供述**：让嫌疑人完整陈述案件事实
4. **初步判断**：评估供述的可信度和自愿性
5. **确认态度**：了解是否愿意认罪认罚

### 6.2 审查性讯问（区别于侦查讯问）

检察官在审查起诉阶段的讯问，与侦查阶段的讯问有本质区别：

| 维度 | 侦查讯问 | 审查起诉讯问 |
|------|---------|-------------|
| **目的** | 获取供述、查明事实 | 核实供述、审查合法性 |
| **立场** | 对抗性较强 | 相对中立 |
| **重点** | 犯罪事实 | 供述自愿性 + 事实核实 |
| **技巧侧重** | 突破心理防线 | 审查判断 + 释法说理 |

### 6.3 针对供述问题的讯问要点

**核实供述自愿性**：
> "你在公安/监委的供述是否属实？是否是你自愿说的？有没有人强迫、威胁或引诱你？"

**核实供述内容**：
> "我再向你确认几个关键事实：第一……第二……你的供述和这些是否一致？"

**核实认罪认罚意愿**：
> "你是否愿意认罪认罚？是否理解认罪认罚的含义和法律后果？你的律师/值班律师是否已经向你提供了法律帮助？"

## 七、实用话术库

### 7.1 讯问开场话术

**正式讯问**：
> "×××，我是×××人民检察院的检察官×××。今天依法对你进行讯问。根据法律规定，你有权委托辩护人，如果你经济困难，可以申请法律援助。你听清楚了吗？"

**告知认罪认罚**：
> "根据《刑事诉讼法》第十五条的规定，犯罪嫌疑人自愿如实供述自己的罪行，承认指控的犯罪事实，愿意接受处罚的，可以依法从宽处理。我现在向你详细解释认罪认罚从宽的含义……"

### 7.2 突破心理防线话术

**证据施压**：
> "你的情况我们已经掌握了充分的证据。你自己说和我们替你说，性质是不一样的。如实供述是从轻处罚的法定情节。"

**同案犯利用**：
> "你的同案×××已经如实供述了全部犯罪事实。你现在如实交代，还能争取从宽。如果和他说的不一致，那谁在说谎，我们一查就清楚。"

**情感突破**：
> "你父母/孩子还在等你回家。你现在如实交代，争取从宽，早点回去，这才是正确的选择。"

**台阶法**：
> "我知道你不是故意要犯罪，这件事确实是一时冲动/法律意识淡薄。你现在如实交代，加上认罪认罚从宽，量刑上会有明显的优惠。"

### 7.3 应对嫌疑人常见反应的话术

**嫌疑人沉默**：
> "你有权保持沉默，但如实供述是法定的从轻情节。我建议你如实回答问题，这对你是有利的。"

**嫌疑人翻供**：
> "你之前的供述有同步录音录像，内容是清楚的。你现在说不属实，请你具体说明哪些不属实，为什么之前那样说。"

**嫌疑人说"不知道""不记得"**：
> "这个事情涉及你的切身利益，我希望你认真回忆。根据其他证据，事实是清楚的。你'不知道'不能改变事实，但如实供述可以改变你的量刑。"

**嫌疑人要求见律师**：
> "你有权委托辩护人。如果你已经委托了律师，我们可以安排会见。如果你没有委托，可以申请法律援助。在律师到来之前，你可以先考虑是否愿意认罪认罚。"

**嫌疑人情绪激动**：
> "我理解你的心情。但情绪解决不了问题，我们是依法办案。你先冷静一下，我们过一会儿再谈。"

### 7.4 认罪认罚协商话术

**达成一致后锁定**：
> "我们刚才协商的内容，我再确认一下：你自愿承认×××的犯罪事实，同意本院提出的判处×××的量刑建议。你的辩护人×××在场并提供了法律帮助。你是否仍然同意？"

**协商陷入僵局**：
> "今天的协商暂时到这里。但提醒你，认罪认罚从宽的量刑优惠是有条件的。如果最终没有达成一致，量刑建议会相应调整。"

**辩护人漫天要价**：
> "你的意见我们理解。但从案件事实和法律规定来看，确实有差距。我们不妨各自把依据说清楚，看看有没有折中的空间。"

## 八、老一辈实务经验提炼

> 以下内容整理自资深检察官和侦查人员的办案经验，结合一线实务素材（贴吧+小红书474条笔记）验证提炼，保留具有实务价值的方法论。

### 8.0 经典讯问方法论

#### "审讯八法"（实务归纳）

| 方法 | 核心操作 | 适用场景 |
|------|---------|---------|
| **直接讯问法** | 开门见山，直指犯罪事实 | 证据确凿、嫌疑人心理防线薄弱时 |
| **迂回讯问法** | 从外围事实逐步切入核心 | 嫌疑人戒备心强、直接讯问容易僵持时 |
| **激将讯问法** | 用刺激性语言激发嫌疑人情绪 | 情绪型嫌疑人、好胜心强者 |
| **引蛇出洞法** | 故意释放虚假信息，引嫌疑人反驳 | 嫌疑人有辩解意愿但不愿主动开口时 |
| **证据突击法** | 在关键时刻出示决定性证据 | 嫌疑人抱有侥幸心理时 |
| **矛盾利用法** | 利用供述中的矛盾反复追问 | 多次供述存在不一致时 |
| **情感感化法** | 以情动人，唤起良知和愧疚 | 家庭观念重、有悔罪可能的嫌疑人 |
| **冷热交替法** | 强硬施压与温和引导交替进行 | 长时间对抗、需要打破僵局时 |

#### "逐步施压法"（首次讯问推荐）

1. **建立 rapport**：先聊基本情况，降低对抗心理
2. **常规提问**：从无争议的事实开始，让嫌疑人"开口说话"
3. **逐步深入**：从外围事实向核心事实推进
4. **施加压力**：在关键问题上放慢节奏，加重语气
5. **出示证据**：在嫌疑人犹豫时出示部分证据
6. **突破防线**：趁势追问，引导如实供述
7. **固定供述**：将供述完整记录，确认签字

#### "九步讯问法"（经典框架）

1. **正面审讯**：直接面对嫌疑人，建立权威
2. **主题发展**：提出犯罪主题，观察反应
3. **阻止否认**：在嫌疑人否认时及时打断并反驳
4. **克服异议**：逐一化解嫌疑人的辩解
5. **获取注意**：让嫌疑人集中注意力听你说
6. **处理消极情绪**：控制嫌疑人的对抗情绪
7. **提出问题**：用封闭式问题引导回答
8. **让嫌疑人陈述**：在突破后让其完整供述
9. **固定供述**：详细记录并确认

### 8.1 "攻心为上"原则

老一辈办案人员的经验总结：**能用心理战解决的，不用物理手段；能用策略突破的，不用蛮力**。

具体体现：
- 讯问前充分研究嫌疑人背景（学历、职业、家庭、性格）
- 根据不同嫌疑人制定个性化讯问方案
- 善用嫌疑人的心理弱点（恐惧、侥幸、牵挂、虚荣）
- 耐心等待心理突破的最佳时机，不急于求成

### 8.2 "先固定后突破"原则

- 让嫌疑人先充分表达其"无罪"或"轻罪"版本
- 完整记录其供述
- 然后逐条用证据反驳
- 使其认识到"抵赖无路"

**价值**：这种方法既固定了供述（防止后续翻供不认），又为突破创造了条件。

### 8.3 "给台阶"原则

- 不要把嫌疑人逼到死角
- 给嫌疑人一个"可以接受"的认罪理由
- 如：法律意识淡薄、一时冲动、受到他人影响等
- 让认罪成为"理性选择"而非"被迫屈服"

**价值**：降低翻供概率，提高供述稳定性。

### 8.4 "证据分层释放"原则

- 不一次性出示全部证据
- 先出示外围证据（到案经过、前科等）
- 再出示直接证据（目击证人、监控等）
- 最后出示核心证据（物证、DNA等）
- 每出示一批证据，给嫌疑人一次"坦白"的机会

**价值**：最大化心理压力，同时保留后续讯问的筹码。

### 8.5 "审讯节奏"原则

- 讯问有张有弛，不能一直紧绷
- 关键突破后适当放缓，让嫌疑人"消化"
- 重要问题反复确认，一般问题快速通过
- 感觉嫌疑人即将突破时，适当暂停，让其自己做决定

## 九、新人学习路径

### 第一阶段：观察学习（入职1-3个月）

- 旁听老检察官的讯问过程，记录其策略和话术
- 学习讯问的基本程序和规则
- 阅读讯问同步录音录像，分析合法策略与非法行为的区别
- 重点阅读：讯问笔录的规范制作

### 第二阶段：辅助实践（入职3-6个月）

- 在老检察官指导下协助准备讯问提纲
- 尝试进行一般案件的首次讯问（在监督下）
- 学习信息控制和矛盾揭露法
- 练习认罪认罚协商的基本流程

### 第三阶段：独立运用（入职6-12个月）

- 独立进行一般案件的审查性讯问
- 在认罪认罚协商中独立与辩护人沟通
- 学会根据嫌疑人类型选择不同的突破策略
- 形成自己的讯问风格

### 第四阶段：持续精进（长期）

- 每起案件复盘讯问得失
- 关注非法证据排除案例，学习边界判断
- 阅读审讯心理学相关著作
- 与同事交流讯问经验

## 十、重要提示

1. **本模块所述策略均以合法为前提**。任何策略的运用不得违反刑事诉讼法的禁止性规定。
2. **讯问的最终目的是查明事实真相**，不是"让嫌疑人认罪"。如果事实真相是嫌疑人无罪，讯问策略应当服务于发现这一真相。
3. **认罪认罚必须建立在自愿性基础上**。任何强迫、威胁、欺骗手段获取的认罪认罚均属违法。
4. **检察官的双重角色**：既是控诉者，也是法律监督者。在讯问中要保持客观公正立场。
5. **保护好自己**：讯问全程同步录音录像，既是保护嫌疑人权利，也是保护检察官自己。
