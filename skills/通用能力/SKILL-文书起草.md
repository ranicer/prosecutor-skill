# 文书起草 Skill

根据案件信息自动生成标准化检察文书，输出 Word 格式文档。

---

## 支持的文书类型

| 文书类型 | 脚本函数 | 说明 |
|---------|---------|------|
| 起诉书 | `create_qishu()` | 公诉案件起诉书 |
| 不起诉决定书 | `create_buqisu()` | 法定/酌定/存疑/附条件不起诉 |
| 审查逮捕意见书 | `create_jianchayijian()` | 逮捕必要性审查 |
| 抗诉书 | - | 对确有错误判决/裁定的抗诉 |
| 检察建议书 | - | 向有关单位提出检察建议 |
| 公益诉讼起诉书 | - | 行政/民事公益诉讼 |
| 纠正违法通知书 | - | 纠正侦查/审判/执行违法 |

---

## 文书生成流程

### 第一步：收集案件信息

根据文书类型，收集对应的信息字段。参考 `SKILL-审查起诉.md` 或 `SKILL-审查逮捕.md` 中的信息收集清单。

### 第二步：填写模板

根据收集的信息，填写对应的文书模板。

模板文件位置：`references/文书模板/`

### 第三步：格式化输出

使用 `scripts/文书格式化.py` 生成 Word 文档：

```python
from scripts.文书格式化 import generate_from_dict

# 准备数据
data = {
    "procuratorate": "×××人民检察院",
    "case_number": "×检×刑诉〔20××〕×号",
    "defendant": {
        "name": "张三",
        "gender": "男",
        "birth_date": "1990年1月1日",
        "id_number": "110101199001011234",
        "ethnicity": "汉",
        "education": "本科",
        "household": "北京市东城区",
        "address": "北京市朝阳区",
        "occupation": "职员"
    },
    "facts": "2024年1月1日...",
    "evidence": ["证据1", "证据2"],
    "legal_basis": "《中华人民共和国刑法》第264条",
    "sentencing_recommendation": "建议判处有期徒刑三年，并处罚金"
}

# 生成文书
output_path = generate_from_dict("起诉书", data, "起诉书.docx")
print(f"文书已生成：{output_path}")
```

## 公文格式规范

| 项目 | 规范 |
|------|------|
| 标题字体 | 方正小标宋_GBK，二号（22pt） |
| 正文字体 | 方正仿宋_GBK，三号（16pt） |
| 小标题字体 | 方正黑体_GBK，三号（16pt） |
| 引用字体 | 方正楷体_GBK，三号（16pt） |
| 行距 | 固定值28磅 |
| 页边距 | 上37mm，下35mm，左28mm，右26mm |
| 首行缩进 | 两个字符（约0.74cm） |

---

## 注意事项

1. **文号规范**：按照检察机关文书编号规则
2. **法条引用**：引用最新有效法律条文
3. **事实描述**：客观、准确、完整
4. **逻辑严密**：事实、证据、法律适用一致
5. **格式统一**：同一案件的文书格式保持一致
