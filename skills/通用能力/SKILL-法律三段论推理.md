# 法律三段论推理 Skill

将法学方法论中的法律三段论（法律规范 → 案件事实 → 法律结论）系统化应用于检察实务，提升案件分析的逻辑严密性。

---

## 一、三段论推理基本框架

```
大前提（Obersatz）：法律规范（法律规定 + 构成要件）
       ↓
小前提（Untersatz）：案件事实（事实认定 + 法律涵摄）
       ↓
结论（Schlussfolgerung）：法律结论（是否符合 / 产生何种法律效果）
```

---

## 二、大前提：法律规范的确定

### 第一步：识别案件涉及的法律领域

| 案件类型 | 主要法律领域 |
|---------|------------|
| 刑事案件 | 刑法 + 刑事诉讼法 + 司法解释 |
| 民事监督案件 | 民法典 + 民事诉讼法 |
| 行政监督案件 | 行政法律法规 + 行政诉讼法 |
| 公益诉讼案件 | 相关实体法 + 检察公益诉讼办案规则 |

### 第二步：确定具体法律规范

**操作**：
1. 从 `references/核心法律/` 中检索相关法条
2. 检索相关司法解释（如"两高"关于×××的解释）
3. 检索指导性案例中的法律适用规则
4. 确认法条为最新有效版本

**输出**：
```
大前提：
《刑法》第×××条第×款第×项规定："……（法条原文）……"
构成要件：
  1. 主体要件：……
  2. 主观要件：……
  3. 客体要件：……
  4. 客观要件：……
```

## 三、小前提：案件事实的涵摄

### 第一步：事实梳理

将案件事实按时间线和逻辑关系梳理：

```
时间线：
- ×年×月×日：……（行为1）
- ×年×月×日：……（行为2）
- ×年×月×日：……（结果发生）

关键事实要素：
- 行为主体：×××
- 行为方式：×××
- 行为对象：×××
- 危害结果：×××
- 因果关系：×××
- 主观心态：×××
```

### 第二步：逐项涵摄

将案件事实逐项与构成要件对照：

```
╔══════════════════════════════════════════════════════════════╗
║                    法 律 涵 摄 对 照 表                       ║
╠════════════════╦══════════════════╦══════════════════════════╣
║ 构成要件        ║ 法律要求          ║ 本案事实                 ║
╠════════════════╬══════════════════╬══════════════════════════╣
║ 犯罪主体        ║ 达到刑事责任年龄   ║ ×年×月×日出生，案发时×岁 ║
║                ║ 具有刑事责任能力   ║ 精神正常，无鉴定意见      ║
╠════════════════╬══════════════════╬══════════════════════════╣
║ 主观方面        ║ 故意/过失        ║                           ║
║                ║ 犯罪目的/动机     ║                           ║
╠════════════════╬══════════════════╬══════════════════════════╣
║ 犯罪客体        ║ 侵犯的法益        ║                           ║
╠════════════════╬══════════════════╬══════════════════════════╣
║ 客观方面        ║ 行为方式          ║                           ║
║                ║ 危害结果          ║                           ║
║                ║ 因果关系          ║                           ║
╠════════════════╬══════════════════╬══════════════════════════╣
║ 涵摄结论        ║ 全部符合/部分符合 ║                           ║
╚════════════════╩══════════════════╩══════════════════════════╝
```

### 第三步：涵摄结论

- **全部符合**：犯罪成立
- **部分不符合**：分析不符合的具体要件，得出不构成犯罪或构成其他犯罪的结论
- **存在争议**：分析争议焦点，提出两种可能的涵摄结论

## 四、结论：法律结论的形成

### 犯罪成立的结论模板

```
三段论分析结论：

大前提：《刑法》第×××条规定了×××罪的构成要件。
小前提：本案中，×××（事实概述），符合×××罪的全部构成要件。
结论：×××的行为构成×××罪，应当依法追究其刑事责任。
```

### 犯罪不成立的结论模板

```
三段论分析结论：

大前提：《刑法》第×××条规定了×××罪的构成要件，要求×××（关键要件）。
小前提：本案中，×××（事实概述），不具有×××（不符合的要件）。
结论：×××的行为不构成×××罪，理由为×××。
```

## 八、使用示例

```
用户：张三和李四在酒吧喝酒，因为座位问题与王五发生口角，
      张三用酒瓶砸王五头部，李四踢了王五两脚，王五倒地后
      张三又踢了头部，王五经抢救无效死亡。帮我分析定性。

助手：[启动三段论推理]

大前提检索：
  候选罪名1：故意伤害罪（致人死亡）— 刑法第234条第2款
  候选罪名2：故意杀人罪 — 刑法第232条

涵摄分析：
  本案关键争议：主观方面是伤害故意还是杀人故意？
  - 酒瓶砸头+踢头部 → 可能认定具有杀人故意
  - 但系因琐事引发、事出有因 → 更符合伤害故意

类案检索：
  检索"酒吧打架致人死亡"类案，×件案例中×件定性故意伤害罪

结论：建议以故意伤害罪（致人死亡）定性
```

## 九、注意事项

1. **大前提必须准确**：法条引用精确到条、款、项，使用最新有效版本
2. **小前提必须有证据支撑**：事实认定必须基于在案证据
3. **涵摄必须逐项对照**：不能跳过任何一个构成要件
4. **结论必须回应所有要件**：即使某个要件显然符合，也要明确写明
5. **争议要件重点论证**：对存在争议的要件进行深入分析
6. **参考司法解释**：两高的司法解释往往对构成要件有具体规定
