# 行政违法行为监督 Skill

协助对行政机关违法行为进行监督，促进依法行政。

---

## 监督范围

1. **违法行使职权**：
   - 行政机关超越法定职权
   - 行政机关违反法定程序
   - 行政机关滥用职权

2. **不作为**：
   - 行政机关不依法履行法定职责
   - 行政机关对违法行为不查处

3. **执法不规范**：
   - 行政执法程序不规范
   - 行政处罚裁量不当
   - 行政许可审批违规

---

## 法律依据

- 《人民检察院组织法》第21条
- 最高检《关于人民检察院对行政违法行为实行法律监督的规定》

---

## 工作流程

1. **线索受理**：通过办案、群众举报、媒体等渠道发现线索
2. **调查核实**：调查行政机关违法行为
3. **提出检察建议**：向行政机关提出纠正违法的检察建议
4. **跟踪落实**：监督检察建议的落实情况

---

## 输出文书

### 行政违法行为监督检察建议书

```

二、法律依据

  - 建议纠正的具体事项
  - 建议期限

请将处理情况书面回复本院。

×××人民检察院
20××年×月×日
```
