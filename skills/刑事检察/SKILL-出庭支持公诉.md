# 出庭支持公诉 Skill

协助检察官完成出庭支持公诉的全部准备工作和庭审应对。

---

## 一、庭前准备 Checklist

### 开庭前 3 天
- [ ] 确认起诉书已送达被告人及辩护人
- [ ] 核实开庭时间、法庭、审判组织（合议庭/独任）
- [ ] 确认辩护人是否提交辩护词/辩护要点
- [ ] 确认被告人是否认罪认罚、是否变更辩护人
- [ ] 准备好全部证据原件（备庭审核查）

### 开庭前 1 天
- [ ] 完成公诉意见书定稿
- [ ] 完成质证提纲（逐项证据对应证明事实）
- [ ] 完成答辩提纲（预判3-5个辩护要点）
- [ ] 检查举证提纲（分组是否合理、顺序是否清晰）
- [ ] 准备好类案检索报告（如辩护人可能质疑量刑）
- [ ] 确认被害人/附带民事诉讼原告是否到庭
- [ ] 准备好法庭教育发言（认罪认罚案件）

### 开庭当天
- [ ] 提前30分钟到庭，检查设备（麦克风、投影）
- [ ] 与书记员确认庭审记录分工
- [ ] 与法官/法官助理确认庭审流程
- [ ] 检查起诉书宣读稿（标注停顿和重点）
- [ ] 准备好质证时出示证据的顺序

## 二、庭审流程与公诉人职责

### 法庭调查阶段

**宣读起诉书**：
- 起立宣读（这是法庭礼仪，不能坐着念）
- 语速适中，关键事实加重语气
- 被告人多人时，逐一宣读各自犯罪事实
- 宣读完毕后确认被告人是否听清

**讯问被告人**：
- 围绕犯罪构成要件提问
- 先封闭式提问固定事实，再开放式提问了解细节
- 被告人翻供时：先宣读此前供述笔录，再结合其他证据质问
- **常见败笔**：问太多开放性问题给被告人表演机会；与辩护人抢话；情绪化追问

**举证质证**：
- 分组举证：按事实分组（如按犯罪事实、量刑情节分组）
- 每组证据说明：证据名称→来源→证明内容
- 质证时简洁回应辩护人质疑，不重复辩护人的话
- 关键证据重点展示（如监控视频、鉴定意见）

### 法庭辩论阶段

**发表公诉意见**：
- 紧扣起诉书指控事实
- 证据分析突出"确实充分"标准
- 定性分析回应辩护人观点
- 量刑建议说明合理性

**答辩要点**：
- 听清辩护人观点再回应，不要预设立场
- 回应要具体，引用证据和法条
- 避免与辩护人情绪化争论
- 遇到新观点：可以请求法庭允许庭后补充书面意见

### 最后阶段
- 审判长询问公诉人有无补充意见
- 确认量刑建议（如庭审中有调整）
- 宣布法庭教育（认罪认罚案件）

## 三、出庭心理建设

### 初任检察官/助理常见焦虑

| 焦虑 | 原因 | 应对 |
|------|------|------|
| 怕忘词 | 庭审紧张大脑空白 | 准备详细的发言提纲，带纸质稿上庭 |
| 怕被辩护人问倒 | 辩护人经验丰富 | 提前预判辩护要点，准备答辩提纲 |
| 怕被法官打断 | 说话太慢或跑题 | 控制发言节奏，紧扣焦点问题 |
| 怕说错话 | 口误可能影响案件 | 关键表述提前写好，照稿宣读 |

### 庭审实用技巧
1. **语速控制**：正常偏慢，关键事实适当加重语气
2. **目光交流**：发言时看审判席，质证时看证据材料
3. **记录习惯**：辩护人发言时记关键词，答辩时对应回应
4. **冷静原则**：被挑衅时不回应，等审判长主持
5. **退庭后**：立即补充庭审笔录，记录辩护人观点和法官态度

## 四、二审出庭专项

### 二审与一审的区别
- 二审是**全面审查**，不限于上诉/抗诉范围
- 检察机关出庭目的是**维护一审正确判决**或**纠正错误判决**
- 辩护人往往在一审基础上提出更深入的辩护意见

### 出庭意见的两种方向

**维持一审判决**（最常见）：
- 一审认定事实清楚，证据确实充分
- 量刑适当，程序合法
- 上诉理由不能成立
- 逐一回应上诉状和辩护意见

**建议改判**（一审确有错误时）：
- 详细指出一审判决的错误之处
- 提出具体的改判建议
- 提交补充证据（如有）
- 注意：需要层报检察长决定

### 二审出庭准备重点
1. 仔细研读上诉状和辩护词
2. 对比一审判决与量刑建议的差距
3. 准备对一审判决的全面审查意见
4. 准备对辩护人新观点的回应

## 五、特殊庭审场景

### 庭审直播/录播
- 注意着装规范（检察制服、检徽佩戴）
- 发言更注意措辞（公众可观看）
- 避免涉及国家秘密、个人隐私的内容
- 语速可适当放慢（方便观众理解）

### 被告人当庭翻供
1. 宣读此前供述笔录（至少2-3份）
2. 指出供述与其他证据的印证关系
3. 追问翻供原因（是否有外部干扰）
4. 请法庭注意翻供的动机和时机
5. 必要时申请休庭核实

### 证人当庭变证
1. 宣读此前证言笔录
2. 询问变证原因
3. 结合其他证据分析证言可信度
4. 如有同步录音录像，可当庭出示

### 被害人/家属情绪激动
1. 保持冷静，不与对方发生冲突
2. 请法庭维持秩序
3. 庭后做好释法说理
4. 必要时启动司法救助告知

## 六、公诉意见书模板

```
审判长、审判员：

根据《中华人民共和国刑事诉讼法》第一百八十六条的规定，
我受×××人民检察院指派，代表本院以国家公诉人的身份，
出席法庭支持公诉，并依法对庭审活动实行法律监督。

现对本案事实、证据及定性量刑发表如下公诉意见：

一、案件事实及证据分析
[对全案证据进行综合分析，说明证据确实充分]

二、案件定性分析
[犯罪构成要件分析、此罪与彼罪分析]

三、量刑意见
[量刑情节分析、量刑建议合理性说明]

四、对辩护意见的回应
[逐一回应辩护人主要辩护观点]

综上所述，请合议庭依法采纳本院的公诉意见。

此致
×××人民法院

公诉人：×××
20××年×月×日
```

## 七、质证提纲模板

```
证据组一：[证明内容，如"犯罪事实"]

证据1：[证据名称]
  来源：[取证机关、时间]
  证明内容：[具体证明的事实]
  质证要点：
    - 合法性：取证程序合法，有[讯问录像/搜查证/扣押清单]佐证
    - 真实性：[内容客观真实，与[其他证据]相互印证]
    - 关联性：[与待证事实直接相关]
  预判质疑及回应：
    辩护人可能质疑：[具体质疑内容]
    回应：[具体回应，引用证据和法律依据]

证据2：...
```

## 注意事项

1. **庭前会议**：参加庭前会议，了解辩护人观点，协商证据出示方式
2. **举证方式**：分组举证，制作举证提纲，明确每组证据证明的事实
3. **法庭辩论**：围绕定罪量刑焦点问题展开，避免偏离主题
4. **庭审记录**：注意记录辩护人的观点和法官的态度
5. **突发情况应对**：准备好对被告人翻供、证人变证等突发情况的应对方案
6. **庭后跟进**：庭审结束后及时补充庭审笔录，向检察官汇报庭审情况
