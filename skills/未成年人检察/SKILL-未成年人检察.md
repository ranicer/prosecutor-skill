# 未成年人检察 Skill

协助办理涉未成年人刑事案件和未成年人权益保护工作。

---

## 特殊原则

1. **教育为主、惩罚为辅**
2. **特殊保护、优先保护**
3. **最有利于未成年人原则**

---

## 核心制度

### 1. 社会调查制度
**法律依据**：刑诉法第279条

- 侦查阶段：公安机关可以进行社会调查
- 审查起诉阶段：人民检察院可以进行社会调查
- 审判阶段：人民法院可以进行社会调查

**社会调查内容**：
1. 未成年人成长经历
2. 家庭环境
3. 受教育情况
4. 犯罪原因
5. 平时表现
6. 社交情况
7. 犯罪后的悔罪表现

### 2. 法律援助强制辩护
**法律依据**：刑诉法第278条

- 犯罪嫌疑人、被告人是未成年人的，没有委托辩护人的，人民法院、人民检察院、公安机关应当通知法律援助机构指派律师为其提供辩护

### 3. 合适成年人到场制度
**法律依据**：刑诉法第281条

- 讯问未成年犯罪嫌疑人、被告人时，法定代理人应当到场
- 法定代理人无法通知、不能到场或者是共犯的，应当通知其他合适成年人到场

### 4. 附条件不起诉
**法律依据**：刑诉法第282条

**适用条件**：
- 未成年人涉嫌刑法分则第四章（侵犯公民人身权利）、第五章（侵犯财产）、第六章（妨害社会管理秩序）规定的犯罪
- 可能判处一年有期徒刑以下刑罚
- 符合起诉条件
- 有悔罪表现

**考验期**：六个月以上一年以下

**考验期义务**：
1. 遵守法律法规，服从监督
2. 按照考察机关的规定报告活动情况
3. 离开居住地需报告
4. 按要求接受矫治和教育

### 5. 犯罪记录封存
**法律依据**：刑诉法第286条

**适用条件**：
- 犯罪时不满十八周岁
- 被判处五年有期徒刑以下刑罚

**封存要求**：
- 犯罪记录予以封存
- 不得向任何单位和个人提供
- 司法机关办案需要或有关单位根据国家规定查询的除外

## 附条件不起诉决定书模板

```
×××人民检察院

附 条 件 不 起 诉 决 定 书

×检×刑未附〔20××〕×号

被不起诉人×××，男/女，×年×月×日出生...

经审查查明：
（未成年人犯罪事实）

本院认为，被不起诉人×××实施了×××行为，涉嫌×××罪，
可能判处一年有期徒刑以下刑罚，符合起诉条件，但有悔罪表现。

根据《中华人民共和国刑事诉讼法》第二百八十二条第一款的
规定，决定对×××附条件不起诉，考验期为×个月。

被不起诉人在考验期内应当遵守下列规定：
1. 遵守法律法规，服从监督
2. 按照考察机关的规定报告自己的活动情况
3. 离开所居住的市、县或者迁居，应当报经考察机关批准
4. 按照考察机关的要求接受矫治和教育

检察员：×××
20××年×月×日
```

---

## 输出文书

1. 社会调查报告
2. 附条件不起诉决定书
3. 考验期满后不起诉决定书
4. 撤销附条件不起诉决定书
5. 犯罪记录封存决定书
6. 未成年人权益保护检察建议

---

## 注意事项

1. **讯问场所**：应当在适合未成年人的场所进行
2. **语言方式**：使用易于理解的语言
3. **心理疏导**：必要时进行心理疏导
4. **家庭参与**：充分听取法定代理人意见
5. **隐私保护**：不得公开未成年人的姓名、住所、照片等
