## 一、快速开始

### 安装位置

Skill 已安装到以下位置（两者同步，任选其一即可）：

| 位置 | 路径 | 说明 |
|------|------|------|
| 用户级 | `~/.claude/skills/prosecutor-assistant/` | 所有项目通用 |
| 项目级 | `.claude/skills/prosecutor-assistant/` | 仅当前项目可用 |

### 生效方式

安装后**重启 Claude Code**即可生效。Skill 会在你提出检察相关问题时自动触发。

## 二、使用方式

### 方式1：直接对话（推荐）

不需要任何特殊命令，直接用自然语言提问即可：

```
你：张三涉嫌盗窃，盗窃金额8000元，有前科，帮我写审查逮捕意见
```

Skill 会自动识别触发，引导你提供案件信息并输出审查意见。

### 方式2：指定文书类型

```
你：帮我起草一份起诉书
你：写不起诉决定书（酌定不起诉）
你：出庭支持公诉，准备质证提纲
```

### 方式3：法律检索

```
你：盗窃罪的数额标准是什么
你：认罪认罚从宽的幅度怎么算
你：帮信罪的构成要件
```

## 四、核心流程：证据整合→量刑→文书（新增v1.2）

针对复杂案件（多罪名、多被告人、证据量大），Skill 提供一体化辅助：

```
你：这个案件3个被告人，涉及盗窃和掩饰隐瞒两个罪名，
    证据材料有80多页，帮我梳理证据并计算量刑。

助手：[启动 SKILL-证据整合与量刑辅助]
1. 结构化提取案件信息（被告人信息、犯罪事实、证据清单）
2. 构建证据矩阵（各证据×各待证事实）
3. 逐项识别量刑情节（从重/从轻/减轻）
4. 分别计算各被告人各罪名的量刑
5. 数罪并罚计算
6. 生成量刑计算表
7. 输出起诉书（Word格式）
```

## 四、文书生成（Word输出）

### 前置条件

```bash
pip install python-docx
```

### 使用方法

提供案件信息后，Skill 会调用 `scripts/文书格式化.py` 生成 Word 文档：

```
你：帮我写起诉书，案件信息如下：
被告人李四，男，1990年5月1日出生，汉族，大学本科，
户籍地北京市海淀区，现住北京市朝阳区，职业：公司职员。
2024年3月1日，李四在北京市朝阳区某商场，趁被害人不注意，
窃取其背包内现金人民币5000元。
证据包括：监控录像、被害人陈述、被告人供述。
认罪认罚，建议判处有期徒刑一年，并处罚金。
```

Skill 会自动整理信息并生成格式规范的 Word 起诉书。

### 文书格式规范

| 项目 | 规范 |
|------|------|
| 标题 | 方正小标宋，二号（22pt），居中 |
| 正文 | 方正仿宋，三号（16pt），首行缩进两字符 |
| 小标题 | 方正黑体，三号（16pt） |
| 行距 | 固定值28磅 |
| 页边距 | 上37mm，下35mm，左28mm，右26mm |

> 注意：如果系统没有安装方正字体，会自动降级为系统默认字体，但格式规范保持不变。

## 五、与 MCP 服务器联动（可选）

如需联网检索法律法规和案例，可配置 MCP 服务器：

### 安装 MCP 服务器

```bash
# 克隆 legal-tools 项目
git clone https://github.com/moyupeng0422/legal-tools.git
cd legal-tools

# 安装依赖
cd 国家法律法规数据库MCP && pip install -r requirements.txt
cd ../人民法院案例库MCP && pip install -r requirements.txt
```

### 启动 MCP 服务器

```bash
# 终端1：启动法律法规数据库（端口18062）
cd legal-tools/国家法律法规数据库MCP/scripts && python server.py

# 终端2：启动人民法院案例库（端口18061）
cd legal-tools/人民法院案例库MCP/scripts && python server.py
```

### 配置 Claude Code MCP

在 Claude Code 设置中添加 MCP 服务器配置：

```json
{
  "mcpServers": {
    "flk-npc": {
      "type": "http",
      "url": "http://localhost:18062/mcp"
    },
    "rmfyalk": {
      "type": "http",
      "url": "http://localhost:18061/mcp"
    }
  }
}
```

### 联动使用

配置 MCP 后，法律检索 skill 会自动使用联网数据库：

```
你：检索"帮助信息网络犯罪活动罪"的最新司法解释
→ 自动调用 flk-npc MCP 检索国家法律法规数据库
→ 输出法律条文和司法解释全文
```

   - 有前科，可能实施新犯罪 ⚠️
   - 涉嫌暴力犯罪 ⚠️

## 七、常见问题

### Q：Skill 没有触发怎么办？

检查：
1. 是否重启了 Claude Code
2. 问题是否涉及检察工作内容
3. 尝试更明确的表述，如"审查逮捕""起诉书"

### Q：生成的 Word 文档格式不对？

确保安装了 python-docx：
```bash
pip install python-docx
```

如果系统没有方正字体，文档会使用默认字体，但格式规范（缩进、行距等）保持不变。

### Q：法条是不是最新的？

Skill 内嵌的法条基于编写时的法律规定（截至2025年）。使用时请自行核实是否有最新修正。如有 MCP 服务器，可通过联网检索获取最新法条。

### Q：能否自定义修改 Skill 内容？

可以。直接编辑 `~/.claude/skills/prosecutor-assistant/` 下的任何文件即可。常用修改：
- 添加新的文书模板到 `references/文书模板/`
- 更新法条到 `references/核心法律/`
- 添加新的子 skill 到 `skills/` 对应目录

### Q：如何与其他 skill 配合使用？

本 skill 与以下 skill 兼容：
- `legal-research-skill`（法律检索）
- `legal-tools`（法律MCP工具）
- 任何其他法律类 skill

## 九、注意事项

1. **本工具输出仅供参考**，不构成正式法律意见
2. **法条引用务必核实**，以最新有效版本为准
3. **案件信息保密**，本工具不会存储或上传任何案件数据
4. **重大案件请以承办检察官判断为准**
5. **建议配合MCP服务器使用**，以获取最新的法律法规和案例
