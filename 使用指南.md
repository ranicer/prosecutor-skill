## 五、模块协作关系

### 审查起诉完整流程

```
案件到达
  ↓
SKILL-常见案型阅卷清单 → 快速定位阅卷重点
  ↓
SKILL-案件分析 → 犯罪构成 + 证据审查 + 量刑分析
  ↓
SKILL-实务操作指南 → 处理退补/沟通/认罪认罚等实务问题
  ↓
SKILL-审查起诉 → 形成审查报告 + 处理决定
  ↓
SKILL-文书起草 → 输出起诉书/不起诉决定书（Word格式）
```

### 出庭支持公诉流程

```
案件分配
  ↓
SKILL-出庭支持公诉（Checklist）→ 庭前准备
  ↓
SKILL-出庭支持公诉（质证提纲+答辩提纲）→ 庭审材料
  ↓
SKILL-实务操作指南 → 庭审突发情况应对
  ↓
SKILL-出庭支持公诉（二审专项）→ 如需二审出庭
```

## 九、常见问题

### Q：Skill 没有触发怎么办？
检查：1. 是否重启了 Claude Code  2. 问题是否涉及检察工作  3. 尝试更明确的表述

### Q：生成的 Word 文档格式不对？
确保安装了 python-docx：`pip install python-docx`

### Q：法条是不是最新的？
基于2025年法律规定，使用时请核实。如有MCP服务器可联网检索最新版本。

### Q：能否自定义修改？
直接编辑 `~/.claude/skills/prosecutor-assistant/` 下的任何文件即可。

## 十、注意事项

1. **本工具输出仅供参考**，不构成正式法律意见
2. **法条引用务必核实**，以最新有效版本为准
3. **案件信息保密**，本工具不会存储或上传任何案件数据
4. **重大案件请以承办检察官判断为准**
5. **建议配合MCP服务器使用**，以获取最新法律法规和案例
6. **数据驱动**：实务内容基于一线分享，定期更新保持实用性
