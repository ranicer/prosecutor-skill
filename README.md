# 检察官实务 Skill

> 中国检察官助理 AI 工具 | Claude Code Skill | 覆盖四大检察全业务条线

一个专为中国检察官助理设计的 Claude Code Skill，覆盖**刑事检察、民事检察、行政检察、公益诉讼检察**全部业务条线，辅助完成案件审查、法律研究、文书起草、出庭准备等工作，输出 **Word 格式**检察文书。

---

## 功能概览

| 模块 | 能力 | 输出文书 |
|------|------|---------|
| **全流程办案** | 从案件受理到归档结案的7阶段完整操作指引 | 全套流程文书 |
| **证据整合与量刑** | 多罪名多被告人证据矩阵、量刑计算、数罪并罚 | 证据矩阵+量刑计算表+起诉书 |
| **审查逮捕** | 逮捕必要性审查、社会危险性评估 | 逮捕审查意见书（Word） |
| **审查起诉** | 全案证据审查、认罪认罚、量刑建议 | 起诉书、不起诉决定书（Word） |
| **出庭支持公诉** | 公诉意见、质证、答辩准备 | 公诉意见书、质证提纲、答辩提纲 |
| **阅卷与出庭技巧** | 结构化阅卷五步法、讯问/举证/辩论话术 | 阅卷笔录、庭审预案 |
| **讯问策略与证据审查** | 讯问心理突破、认罪认罚协商、供述审查、非法证据识别 | 讯问提纲、具结书（v2.2新增） |
| **检察官主动权与对抗性** | 对公安施压、对律师反制、对法院抗诉、法庭辩论进攻 | 答辩提纲、纠正违法通知书（v2.3新增） |
| **民事检察** | 裁判监督、执行监督 | 再审检察建议书、抗诉书 |
| **行政检察** | 裁判监督、违法行为监督 | 检察建议书 |
| **公益诉讼** | 行政/民事公益诉讼 | 诉前检察建议、起诉书 |
| **未成年人检察** | 附条件不起诉、犯罪记录封存 | 附条件不起诉决定书 |
| **控告申诉** | 控告申诉审查、国家赔偿 | 国家赔偿决定书 |
| **实务操作指南** | 与公安/法院/被害人沟通应对 | 最佳处理方案和话术 |
| **职业发展与竞赛** | 员额晋升、公诉人竞赛备考、职场规范 | 竞赛备考指南 |

## 安装

### 方式一：手动安装

```bash
cd ~/.claude/skills
git clone https://github.com/ranicer/prosecutor-skill.git prosecutor-assistant
```

重启 Claude Code 即可生效。

### 方式二：复制安装

```bash
cp -r prosecutor-skill ~/.claude/skills/prosecutor-assistant
```

### 生成 Word 文书（可选）

```bash
pip install python-docx
```

## 使用示例

### 审查逮捕

```
你：公安机关提请批准逮捕，犯罪嫌疑人张三涉嫌盗窃罪，
    盗窃金额8000元，有前科（盗窃罪判一年），现被刑事拘留。

助手：[自动触发审查逮捕skill]
1. 犯罪构成分析：盗窃罪（刑法第264条）
2. 证据审查：现有证据确实充分
3. 社会危险性评估：
   - 有前科，可能实施新犯罪 ⚠️
   - 综合评估：有社会危险性
4. 处理意见：建议批准逮捕

→ 输出：审查逮捕意见书.docx
```

### 审查起诉 + 起诉书

```
你：帮我写起诉书，案件信息如下：
    被告人李四，男，1990年出生...
    [案件事实和证据]

助手：[Full模式]
→ 输出：起诉书.docx（公文格式）
```

### 实务问题

```
你：退补后公安补查材料不全怎么办？
你：被告人签了认罪认罚具结书后翻供了怎么办？
你：法院不采纳量刑建议怎么应对？
```

### 讯问与协商（v2.2新增）

```
你：嫌疑人一直不认罪，态度很硬，怎么突破？
你：认罪认罚协商陷入僵局了怎么办？
你：怎么审查讯问的合法性？
```

### 检察官主动权（v2.3新增）

```
你：辩护人要做无罪辩护，我怎么准备答辩？
你：法院判决没有采纳我的量刑建议，差了很多，怎么办？
你：公安补查材料又敷衍了，怎么推进？
```

## MCP 服务器集成（可选）

如需联网检索法律法规和案例，可配置 MCP 服务器：

```json
{
  "mcpServers": {
    "flk-npc": {
      "type": "http",
      "url": "http://localhost:18062/mcp"
    },
    "rmfyalk": {
      "type": "http",
      "url": "http://localhost:18061/mcp"
    }
  }
}
```
