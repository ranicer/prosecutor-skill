# 缓刑适用指引（检察工作用）

> 供检察官助理在提出缓刑量刑建议时参考。

---

## 一、缓刑适用条件

### 法律依据

**《刑法》第72条**：
被判处拘役、三年以下有期徒刑的犯罪分子，同时符合下列条件的，可以宣告缓刑，对其中不满十八周岁的人、怀孕的妇女和已满七十五周岁的人，应当宣告缓刑：
1. 犯罪情节较轻
2. 有悔罪表现
3. 没有再犯罪的危险
4. 宣告缓刑对所居住社区没有重大不良影响

### 禁止适用缓刑的情形

1. **累犯**（刑法第74条）：累犯不适用缓刑
2. **犯罪集团首要分子**（刑法第74条）：犯罪集团的首要分子不适用缓刑
3. **数罪并罚后刑期超过3年**的

## 二、缓刑适用评估要素

### 犯罪情节较轻

| 评估要素 | 从轻考量 | 从重考量 |
|---------|---------|---------|
| 犯罪性质 | 非暴力犯罪 | 暴力犯罪 |
| 犯罪数额/后果 | 数额较小/后果较轻 | 数额巨大/后果严重 |
| 犯罪手段 | 一般手段 | 特别残忍手段 |
| 犯罪动机 | 生活所迫/临时起意 | 预谋/报复/贪利 |
| 被害人过错 | 被害人有明显过错 | 被害人无过错 |

### 有悔罪表现

- 自首、坦白
- 自愿认罪认罚
- 积极赔偿被害人损失
- 取得被害人谅解
- 退赃退赔
- 当庭自愿认罪

### 没有再犯罪的危险

| 评估要素 | 再犯风险低 | 再犯风险高 |
|---------|----------|----------|
| 前科劣迹 | 无前科 | 有前科/累犯 |
| 犯罪类型 | 偶发性犯罪 | 习惯性/职业性犯罪 |
| 个人情况 | 有固定住所和工作 | 无固定住所和工作 |
| 家庭支持 | 家庭关系良好 | 家庭关系破裂 |
| 犯罪后表现 | 积极改造 | 拒不悔改 |

### 宣告缓刑对社区无重大不良影响

- 社区矫正机构出具的调查评估意见
- 被害人及社区居民的态度
- 案件的社会影响

---

## 三、社区矫正调查评估

### 评估内容

1. 被告人的基本情况
2. 居住条件和家庭情况
3. 工作单位和收入情况
4. 社区矫正的监管条件
5. 社区居民的意见
6. 是否具备帮教条件

### 评估意见对缓刑建议的影响

- **适合社区矫正**：支持缓刑建议
- **不完全具备条件**：需谨慎，综合考虑
- **不具备社区矫正条件**：不建议缓刑

## 四、缓刑量刑建议模板

```
量刑建议：
一、主刑：有期徒刑×年×个月（缓刑×年×个月）
二、缓刑考验期：×年×个月至×年×个月

缓刑适用理由：
1. 犯罪情节较轻：×××
2. 有悔罪表现：×××（认罪认罚/赔偿/谅解等）
3. 没有再犯罪的危险：×××（无前科/有固定住所/家庭支持等）
4. 宣告缓刑对所居住社区没有重大不良影响：×××
5. 社区矫正机构评估意见：×××

不符合缓刑条件的情形排除：
- 非累犯：经查询无前科劣迹
- 非犯罪集团首要分子
- 数罪并罚后刑期不超过三年
```

## 五、常见罪名缓刑适用参考

| 罪名 | 缓刑适用要点 |
|------|------------|
| 盗窃罪 | 数额较大+认罪认罚+退赃+初犯→可建议缓刑 |
| 故意伤害罪（轻伤） | 轻伤二级+赔偿谅解+初犯→可建议缓刑 |
| 危险驾驶罪 | 血液酒精含量不高+认罪认罚→通常建议缓刑 |
| 交通肇事罪 | 积极赔偿+谅解+有悔罪表现→可建议缓刑 |
| 诈骗罪 | 数额较大+退赔+认罪认罚→可建议缓刑 |
| 交通肇事罪（逃逸） | 逃逸情节→一般不建议缓刑 |
| 故意伤害罪（重伤） | 重伤→一般不建议缓刑 |

## 六、注意事项

1. **量刑建议与判决的关系**：量刑建议可以包含缓刑建议，但最终由法院决定
2. **社区矫正评估前置**：建议缓刑前应了解社区矫正机构的评估意见
3. **被害人意见**：缓刑建议应考虑被害人态度，被害人强烈反对的需慎重
4. **同案犯比较**：同案犯中主犯、从犯的缓刑适用应有所区别
5. **附条件**：可以根据案件情况建议缓刑考验期的禁止令
