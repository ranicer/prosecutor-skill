# 常见犯罪量刑指引（检察工作用）

> 本指引基于最高人民法院《关于常见犯罪的量刑指导意见》及相关规范性文件整理，
> 供检察官助理在量刑建议工作中参考。具体量刑应结合案件实际情况和当地执行标准。

---

## 一、量刑的基本方法

### 量刑步骤
1. **确定量刑起点**：根据基本犯罪构成事实，在相应的法定刑幅度内确定量刑起点
2. **确定基准刑**：根据其他影响犯罪构成的犯罪事实，在量刑起点的基础上增加刑罚量确定基准刑
3. **确定宣告刑**：根据量刑情节调节基准刑，并综合考虑全案情况，依法确定宣告刑

### 量刑情节调节比例

| 量刑情节 | 调节比例 |
|---------|---------|
| 自首 | 可以减少基准刑的40%以下；犯罪较轻的，可以减少40%以上或免除处罚 |
| 坦白 | 可以减少基准刑的20%以下；如实供述避免严重后果的，可减少30%以下 |
| 立功 | 可以减少基准刑的20%以下；重大立功的，可以减少30%-50% |
| 当庭自愿认罪 | 可以减少基准刑的10%以下 |
| 认罪认罚 | 从宽幅度一般不超过基准刑的30% |
| 取得谅解 | 可以减少基准刑的40%以下；积极赔偿并取得谅解的，可减少40%以上 |
| 积极赔偿但未取得谅解 | 可以减少基准刑的30%以下 |
| 初犯、偶犯 | 可以减少基准刑的10%-30% |
| 累犯 | 增加基准刑的10%-40% |
| 前科劣迹 | 增加基准刑的10%以下 |

### 认罪认罚从宽幅度

| 认罪认罚阶段 | 从宽幅度 |
|-------------|---------|
| 侦查阶段认罪认罚 | 可以减少基准刑的30%以下 |
| 审查起诉阶段认罪认罚 | 可以减少基准刑的20%以下 |
| 审判阶段认罪认罚 | 可以减少基准刑的15%以下 |

## 二、常见犯罪量刑

### （一）交通肇事罪

**第一档：三年以下有期徒刑或拘役**
- 死亡1人或重伤3人以上，负全部或主要责任
- 死亡3人以上，负同等责任
- 造成公共/他人财产直接损失，无能力赔偿30万元以上

**第二档：三年以上七年以下有期徒刑**
- 交通运输肇事后逃逸
- 有其他特别恶劣情节（死亡2人以上或重伤5人以上，负全部或主要责任等）

**第三档：七年以上有期徒刑**
- 因逃逸致人死亡

**量刑起点**：
- 致1人重伤，负全部或主要责任：6个月-1年6个月有期徒刑
- 致1人死亡，负全部或主要责任：1年-2年有期徒刑
- 逃逸：3年-4年有期徒刑
- 逃逸致死：7年-8年有期徒刑

### （二）故意伤害罪

**第一档：三年以下有期徒刑、拘役或管制**
- 故意伤害致1人轻伤

**第二档：三年以上十年以下有期徒刑**
- 故意伤害致1人重伤

**第三档：十年以上有期徒刑、无期徒刑或死刑**
- 故意伤害致1人死亡
- 以特别残忍手段致1人重伤造成严重残疾

**量刑起点**：
- 致1人轻伤：6个月-1年6个月有期徒刑
- 致1人重伤：3年-5年有期徒刑
- 以特别残忍手段致1人重伤造成六级严重残疾：10年-13年有期徒刑

**增加刑罚量**：
- 每增加1人轻微伤：增加1-2个月刑期
- 每增加1人轻伤：增加3-6个月刑期
- 每增加1人重伤：增加1-2年刑期

### （三）盗窃罪

**数额较大（3千-10万元）：三年以下有期徒刑、拘役或管制**

量刑起点：
- 数额较大：1年以下有期徒刑、拘役
- 多次盗窃、入户盗窃、携带凶器盗窃、扒窃：6个月-1年有期徒刑

**数额巨大（3万-50万元）：三年以上十年以下有期徒刑**

量刑起点：3年-4年有期徒刑

**数额特别巨大（30万-500万元）：十年以上有期徒刑或无期徒刑**

量刑起点：10年-12年有期徒刑

**增加刑罚量**：
- 数额每增加5000元：增加1-2个月刑期（数额较大档）
- 数额每增加5万元：增加3-6个月刑期（数额巨大档）

### （四）诈骗罪

**数额较大（3千-10万元）：三年以下有期徒刑、拘役或管制**

量刑起点：6个月-1年有期徒刑

**数额巨大（3万-50万元）：三年以上十年以下有期徒刑**

量刑起点：3年-4年有期徒刑

**数额特别巨大（50万元以上）：十年以上有期徒刑或无期徒刑**

量刑起点：10年-12年有期徒刑

### （五）抢劫罪

**基本犯（三年以上十年以下有期徒刑）**

量刑起点：3年-5年有期徒刑

**加重犯（十年以上有期徒刑、无期徒刑或死刑）**

量刑起点：10年-13年有期徒刑

**增加刑罚量**：
- 每增加1次抢劫：增加6个月-1年刑期
- 每增加1人轻微伤：增加1-2个月刑期
- 每增加1人轻伤：增加3-6个月刑期
- 每增加1人重伤：增加1-2年刑期
- 每增加1人死亡：增加2-3年刑期

### （六）寻衅滋事罪

**五年以下有期徒刑、拘役或管制**

量刑起点：
- 随意殴打他人，情节恶劣：6个月-1年有期徒刑
- 追逐拦截辱骂恐吓他人，情节恶劣：6个月-1年有期徒刑
- 强拿硬要或任意损毁占用公私财物，情节严重：6个月-1年有期徒刑
- 在公共场所起哄闹事，造成严重混乱：6个月-1年有期徒刑

**纠集他人多次实施前款行为，严重破坏社会秩序（五年以上十年以下有期徒刑）**

量刑起点：5年-7年有期徒刑

### （七）危险驾驶罪

**拘役，并处罚金**

量刑起点：
- 血液酒精含量80-140mg/100ml：1-2个月拘役
- 血液酒精含量140-200mg/100ml：2-3个月拘役
- 血液酒精含量200mg/100ml以上：3-4个月拘役

**增加刑罚量**：
- 血液酒精含量每增加50mg/100ml：增加15-30日刑期
- 造成交通事故且负全部或主要责任：增加1-2个月刑期
- 在高速公路、城市快速路上驾驶：增加1-2个月刑期
- 驾驶载有乘客的营运机动车：增加1-2个月刑期

**罚金**：
- 数额标准：一般不少于1000元
- 根据醉酒程度、是否造成后果、认罪悔罪态度等确定

### （八）帮助信息网络犯罪活动罪

**三年以下有期徒刑或拘役，并处或单处罚金**

量刑起点：6个月-1年有期徒刑

**增加刑罚量**：
- 支付结算金额每增加50万元：增加2-3个月刑期
- 为对象数量每增加3个：增加1-2个月刑期
- 违法所得每增加5000元：增加1-2个月刑期

### （九）掩隐罪（掩饰、隐瞒犯罪所得罪）

**第一档：三年以下有期徒刑、拘役或管制，并处或单处罚金**

量刑起点：6个月-1年有期徒刑

**第二档：三年以上七年以下有期徒刑，并处罚金**

量刑起点：3年-4年有期徒刑

**增加刑罚量**：
- 犯罪数额每增加5000元：增加1-2个月刑期（第一档）
- 犯罪数额每增加5万元：增加3-6个月刑期（第二档）

## 三、缓刑适用条件

### 可以适用缓刑
- 被判处拘役、三年以下有期徒刑
- 犯罪情节较轻
- 有悔罪表现
- 没有再犯罪的危险
- 宣告缓刑对所居住社区没有重大不良影响

### 应当适用缓刑
- 不满18周岁的人
- 怀孕的妇女
- 已满75周岁的人

### 一般不适用缓刑的情形
- 累犯和犯罪集团的首要分子
- 犯罪后果严重、社会影响恶劣的
- 曾因同类行为受过刑事处罚或行政处罚的
- 认罪态度不好、无悔罪表现的

## 四、免予刑事处罚的适用

犯罪情节轻微不需要判处刑罚的，可以免予刑事处罚。具有下列情形之一的，一般可以免予刑事处罚：
1. 又聋又哑的人或者盲人犯罪
2. 防卫过当
3. 避险过当
4. 犯罪预备
5. 犯罪中止
6. 从犯
7. 胁从犯
8. 自首并且犯罪较轻的
9. 大功以上立功的
10. 其他情节轻微不需要判处刑罚的

## 五、量刑建议注意事项

1. **确定刑量刑建议**：认罪认罚案件，一般应当提出确定刑量刑建议
2. **幅度刑量刑建议**：非认罪认罚案件或特殊情况，可以提出幅度刑量刑建议
3. **量刑建议说理**：应当在量刑建议书中说明量刑理由和依据
4. **听取意见**：提出量刑建议前，应当听取犯罪嫌疑人、辩护人或值班律师的意见
5. **调整量刑建议**：审判中出现新的量刑情节或辩护方提出合理异议的，可以调整量刑建议
