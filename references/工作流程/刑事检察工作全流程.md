# 刑事检察工作全流程详解

> 本流程涵盖从公安机关立案到判决执行的全部刑事检察环节，
> 标注每个环节的期限、关键节点和实务要点。

---

## 一、审查逮捕阶段

```
公安机关提请批准逮捕
       ↓
检察机关受理（当日）
       ↓
  ┌── 讯问犯罪嫌疑人（有疑问/要求当面陈述/侦查可能违法的，必须讯问）
  ├── 听取辩护律师意见（辩护律师提出要求的，必须听取）
  └── 审查证据材料
       ↓
  ┌─────────────────────────────┐
  ↓              ↓              ↓
批准逮捕    不批准逮捕    需补充侦查
（公安执行） （公安释放/变更） （退回公安，通知公安）
       ↓
执行逮捕后24小时内通知家属
```

**期限**：审查批捕期限为 **7日**

**实务要点**：
1. 不批准逮捕的，必须说明理由
2. 不批捕且需要补充侦查的，同时通知公安机关
3. 批捕后羁押期限侦查阶段一般不超过2个月

---

## 二、审查起诉阶段

```
公安机关移送审查起诉
       ↓
检察机关受理
       ↓
形式审查（材料是否齐全）
  ↓ 材料不全 → 要求补充材料
       ↓
全面阅卷审查
       ↓
讯问犯罪嫌疑人（必须讯问）
       ↓
听取被害人及其诉讼代理人意见
       ↓
听取辩护人或值班律师意见
       ↓
审查判断（是否达到起诉标准）
       ↓
┌─────────┬─────────────┬──────────────┐
↓         ↓             ↓              ↓
提起公诉  不起诉        退回补充侦查    自行补充侦查
（移送法院）（法定/酌定/   （每次1个月，  （每次1个月，
           存疑不起诉）   最多2次）      无次数限制）
```

**期限**：
- 一般审查起诉期限：**1个月**
- 重大复杂案件可延长 **15日**
- 速裁案件：**10日**（可延长至15日）
- 退补：每次 **1个月**，最多 **2次**

**认罪认罚案件特殊流程**：
```
告知认罪认罚权利和法律后果
       ↓
听取辩护人/值班律师意见
       ↓
犯罪嫌疑人签署认罪认罚具结书
（辩护人/值班律师在场并签字）
       ↓
提出量刑建议（一般确定刑）
       ↓
建议适用速裁程序/简易程序（符合条件时）
```

**实务要点**：
1. 审查起诉阶段必须讯问犯罪嫌疑人
2. 辩护律师提出要求的，必须听取意见
3. 被害人意见也应当听取
4. 退补以2次为限，不能反复退补
5. 认罪认罚具结书签署时应全程录音录像

## 三、出庭支持公诉阶段

```
法院通知开庭
       ↓
庭前准备
  ├── 阅卷（补充阅卷）
  ├── 准备出庭材料
  │   ├── 公诉意见书
  │   ├── 举证提纲
  │   ├── 质证提纲
  │   ├── 答辩提纲
  │   └── 讯问提纲
  ├── 庭前会议（如有）
  └── 多媒体示证材料准备
       ↓
┌─────── 法庭庭审 ───────────────┐
│                                 │
│  开庭（查明当事人到庭、宣布案由） │
│       ↓                         │
│  法庭调查                       │
│  ├── 公诉人宣读起诉书            │
│  ├── 被告人陈述                  │
│  ├── 讯问被告人                  │
│  ├── 举证质证                    │
│  │   ├── 物证、书证              │
│  │   ├── 证人证言               │
│  │   ├── 被害人陈述              │
│  │   ├── 鉴定意见               │
│  │   └── 其他证据               │
│  └── 控辩双方对证据和事实发表意见  │
│       ↓                         │
│  法庭辩论                       │
│  ├── 公诉人发表公诉意见           │
│  ├── 辩护人发表辩护意见           │
│  └── 控辩双方互相辩论            │
│       ↓                         │
│  被告人最后陈述                  │
│       ↓                         │
│  休庭评议/当庭宣判              │
└─────────────────────────────────┘
       ↓
判决后：
  ├── 无抗诉必要 → 审判监督（判决生效后监督）
  ├── 有抗诉必要 → 10日内提出抗诉
  └── 被害人不服 → 引导申诉/自诉
```

**期限**：
- 一审普通程序：受理后 **2个月**以内宣判，至迟不超过 **3个月**
- 可延长 **3个月**（特殊情况经批准）
- 速裁程序：**10日**以内（可能判处1年以上的，15日）

---

## 四、二审阶段

```
被告人/自诉人上诉 / 检察机关抗诉
       ↓
二审法院受理
       ↓
┌── 开庭审理（应当开庭的情形）──────┐
│  ├── 被告人、自诉人对事实、证据提出异议 │
│  ├── 可能影响定罪量刑的上诉案件       │
│  ├── 被告人被判处死刑的上诉案件       │
│  └── 人民检察院抗诉的案件             │
└───────────────────────────────────┘
       ↓
┌── 书面审理（其他案件）──────────────┐
│  合议庭阅卷、调查核实、提讯被告人      │
│  必要时可以听取辩护人意见             │
└───────────────────────────────────┘
       ↓
判决：
  ├── 维持原判
  ├── 依法改判
  └── 撤销原判，发回重审
```

**期限**：
- 二审期限：**2个月**以内审结
- 可延长 **2个月**（特殊情况经批准）

**检察机关二审出庭**：
- 检察机关抗诉的案件，检察院应派员出庭
- 上诉案件，检察院可以出庭也可以不出庭

---

## 五、审判监督阶段

```
当事人申诉 / 检察机关发现错误
       ↓
审查是否符合再审条件
  ├── 有新证据足以推翻原判决
  ├── 认定事实确有错误
  ├── 适用法律确有错误
  ├── 审判程序严重违法
  └── 审判人员贪污受贿、枉法裁判
       ↓
  ┌──────────────────────┐
  ↓                      ↓
法院决定再审          检察机关提出抗诉
```

**检察机关抗诉**：
- 最高检对各级法院、上级检对下级法院已生效判决裁定
- 发现确有错误的，按审判监督程序向同级法院提出抗诉
- 不受时间限制

---

## 六、刑罚执行监督阶段

```
判决生效后交付执行
       ↓
┌──────────┬──────────┬──────────┐
↓          ↓          ↓          ↓
有期徒刑   无期徒刑   死刑      管制/拘役
（监狱执行）（监狱执行）（最高法核准） （社区矫正）
       ↓          ↓
减刑/假释审查 → 检察机关监督
       ↓
暂予监外执行审查 → 检察机关监督
       ↓
刑罚执行完毕 → 释放检察
```

## 七、期限速查表

| 环节 | 期限 | 可延长 | 法律依据 |
|------|------|--------|---------|
| 审查批捕 | 7日 | 不可延长 | 刑诉法第87条 |
| 侦查羁押（一般） | 2个月 | 可延长1-2个月 | 刑诉法第156条 |
| 审查起诉（一般） | 1个月 | 重大复杂延长15日 | 刑诉法第172条 |
| 审查起诉（速裁） | 10日 | 可延长至15日 | 刑诉法第172条 |
| 退补侦查 | 1个月 | 以2次为限 | 刑诉法第175条 |
| 一审普通程序 | 2个月 | 至迟3个月，可再延3个月 | 刑诉法第208条 |
| 一审速裁程序 | 10日 | 可延长至15日 | 刑诉法第225条 |
| 二审 | 2个月 | 可延长2个月 | 刑诉法第243条 |
| 抗诉期限 | 10日 | 不可延长 | 刑诉法第230条 |
