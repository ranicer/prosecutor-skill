# 类案检索工作流程

> 检察机关类案检索的标准操作流程，供检察官助理在案件审查和出庭准备中使用。

---

## 一、启动类案检索的条件

### 应当检索的情形

1. **定性争议案件**：此罪与彼罪存在争议
2. **量刑建议有分歧**：与法院/辩护人量刑意见差异较大
3. **新类型案件**：首次遇到或少见的案件类型
4. **重大敏感案件**：社会关注度高、舆论关注度高的案件
5. **法律适用有争议**：法律规定不够明确、存在理解分歧

### 可以检索的情形

1. 认罪认罚量刑协商需要参考
2. 不起诉决定需要论证
3. 出庭准备需要预案
4. 业务竞赛/论辩赛积累案例

---

## 二、检索来源与优先级

```
第一优先：最高检指导性案例（应当参照）
    ↓
第二优先：人民法院案例库（指导性案例+参考案例）
    ↓
第三优先：本院/本省同类案件（参考价值最高）
    ↓
第四优先：中国裁判文书网（大量案例）
    ↓
补充来源：最高检公报案例、典型案例
```

---

## 三、检索操作步骤

### Step 1：确定检索关键词

| 案件要素 | 关键词提取方法 |
|---------|--------------|
| 罪名 | 罪名全称 + 罪名简称 |
| 犯罪手段 | 具体手段描述（如"酒瓶砸头""持刀"） |
| 犯罪数额 | 具体金额区间（如"3万-5万"） |
| 犯罪后果 | 具体后果（如"轻伤二级""死亡"） |
| 量刑情节 | 自首/立功/认罪认罚/累犯/初犯等 |
| 争议焦点 | 此罪彼罪对比关键词 |

### Step 2：选择检索平台

| 平台 | 适用场景 | 检索方式 |
|------|---------|---------|
| 最高检官网 | 指导性案例 | 按编号/按罪名/按领域 |
| 人民法院案例库 | 指导性案例+参考案例 | MCP 或网页检索 |
| 中国裁判文书网 | 大量案例 | 按案由/关键词/法院/时间 |
| 北大法宝 | 深度法律研究 | 按法条/案例/论文 |

### Step 3：筛选与对比

**筛选标准**：
- 罪名相同或相似
- 犯罪手段/数额/后果相近
- 量刑情节相似
- 裁判时间在近两年

**对比分析**：
- 使用"类案对比分析表"进行逐项对比
- 分析本案与参考案例的相同点和差异点
- 论证参考案例对本案的适用性

### Step 4：输出检索报告

```
类案检索报告

一、检索背景
  案件名称：
  检索原因：（定性争议/量刑分歧/出庭准备）
  检索时间：

二、检索过程
  检索关键词：
  检索平台：
  检索结果数量：

三、参考案例

  案例1：
    案号：
    罪名：
    事实摘要：
    裁判结果：
    参考价值：

  案例2：
    ...

四、类案对比分析

  | 比较维度 | 本案 | 参考案例1 | 参考案例2 |
  |---------|------|----------|----------|
  | 罪名 | | | |
  | 犯罪手段 | | | |
  | 犯罪数额/后果 | | | |
  | 主要量刑情节 | | | |
  | 处理结果 | | | |

五、检索结论
  1. 同类案件的普遍做法
  2. 本案与同类案件的差异
  3. 对本案处理的参考建议
```

---

## 四、检索报告在检察工作中的运用

### 审查起诉阶段

- 定性有争议时，参考同类案件的定性做法
- 决定是否起诉时，参考同类案件的处理结果
- 认罪认罚量刑协商时，参考同类案件的从宽幅度

### 出庭支持公诉阶段

- 准备《类案检索报告》作为庭审材料
- 回应辩护人"量刑过重"时引用同类案件量刑
- 回应"此罪彼罪"争议时引用定性先例
- 准备法官可能追问的类案参考

### 认罪认罚协商阶段

- 向辩护人展示同类案件的量刑结果
- 作为量刑建议合理性的论据
- 增强量刑协商的说服力

---

## 五、注意事项

1. **全面检索**：不能只检索支持自己观点的案例，也要检索相反观点
2. **优先效力**：指导性案例效力最高，同辖区案例参考价值更高
3. **时效性**：优先检索近两年案例，注意法律修改的影响
4. **差异分析**：必须分析本案与参考案例的差异，不能简单类比
5. **保密义务**：检索到的案件信息属于检察工作秘密
6. **持续积累**：将检索到的有价值案例归档，形成案例库
