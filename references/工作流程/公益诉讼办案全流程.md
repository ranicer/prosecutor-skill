# 公益诉讼办案全流程

---

## 一、行政公益诉讼流程

```
线索发现
├── 办案中发现
├── 群众举报
├── 媒体发现
├── 上级交办
└── 其他机关移送
    ↓
线索评估（7日内）
├── 是否属于监督范围
├── 是否存在公益损害
├── 行政机关是否违法
└── 是否属于本院管辖
    ↓
立案审批
    ↓
调查取证
├── 制作调查方案
├── 收集书证、物证
├── 询问相关人员
├── 委托鉴定/检测
├── 现场勘查
└── 必要时委托审计
    ↓
调查终结报告
    ↓
诉前检察建议
├── 检察长审批
├── 制作检察建议书
├── 送达行政机关
└── 抄送同级人大
    ↓
跟踪监督（2个月）
├── 行政机关回复
├── 审查整改情况
└── 开展"回头看"
    ↓
┌────────────────────────────┐
↓                            ↓
已整改到位                  未整改到位
→ 终结审查                  → 提起诉讼
→ 制作结案报告              → 出庭支持诉讼
```

## 二、民事公益诉讼流程

```
线索发现 → 评估 → 立案 → 调查取证
    ↓
诉前公告（省级以上媒体，30日）
    ↓
公告期满
    ↓
┌────────────────────────────┐
↓                            ↓
法律规定的机关              无适格主体
或有关组织已起诉            → 检察机关起诉
→ 支持起诉                  → 制作起诉书
```

## 三、关键期限速查

| 环节 | 期限 |
|------|------|
| 线索评估 | 7日内 |
| 检察建议回复 | 2个月 |
| 诉前公告 | 30日 |
| 法院审理（一审） | 6个月 |
